# DS1302 执行时序图与执行过程

## 一、写操作时序

```
RST  ─────┐        ┌─────────────────────┐        ┌────
          └────────┘                     └────────┘
      
SCLK ─────────┐  ┌──┐  ┌──┐  ┌──┐  ┌──┐  ┌──┐  ┌──
              └──┘  └──┘  └──┘  └──┘  └──┘  └──┘
              ↑
              SCLK上升沿写入
          
IO   ────D0───┴──D1───────┴──D2───────┴──D3───────┴──
          
时序：RST拉高 → 8个SCLK发地址 → 8个SCLK发数据 → RST拉低
```

**关键点**：

- RST高电平使能通信
- 先设置IO值，再SCLK=1（上升沿写入）
- 数据从**低位到高位**依次发送（LSB first）

---

## 二、读操作时序

```
RST  ─────┐                          ┌────
          └──────────────────────────┘
      
SCLK ─────────┐  ┌──┐  ┌──┐  ┌──┐  ┌──
              └──┘  └──┘  └──┘  └──┘
                    ↓
                    SCLK下降沿后采样
                
IO（DS1302）────────┴─D0────────┴─D1──
```

**关键点**：

- 先发地址（告诉DS1302读哪个寄存器）
- SCLK下降沿后，DS1302输出数据到IO线
- 单片机在SCLK上升沿采样IO线上的数据

## 三、函数一：Write_Ds1302 - 底层单字节写入

### 代码实现

```c
void Write_Ds1302(unsigned char temp)
{
    unsigned char i;
    for (i = 0; i < 8; i++) // 循环8次，每次写入1位
    {
        SCK = 0;           // SCK拉低，准备放置数据
        SDA = temp & 0x01; // 取出temp的最低位（bit0），放到SDA数据线上
        temp >>= 1;        // temp右移1位，为下一次循环准备下一位
        SCK = 1;           // SCK拉高，DS1302采样SDA上的数据
    }
}
```

### 执行过程示例：发送 `0x84`（写小时寄存器的地址）

```
初始：temp = 0x84 = 1000 0100（二进制）

第1轮：
    SCK = 0                    // 时钟拉低，准备放数据
    SDA = temp & 0x01 → 1000 0100 & 0000 0001 = 0 → SDA=0（发送bit0）
    temp >>= 1  → temp = 0100 0010
    SCK = 1                    // 时钟拉高，数据被采样

第2轮：
    SCK = 0
    SDA = temp & 0x01 → 0100 0010 & 0000 0001 = 0 → SDA=0（发送bit1）
    temp >>= 1  → temp = 0010 0001
    SCK = 1

第3轮：
    SCK = 0
    SDA = temp & 0x01 → 0010 0001 & 0000 0001 = 1 → SDA=1（发送bit2）
    temp >>= 1  → temp = 0001 0000
    SCK = 1

第4轮：
    SCK = 0
    SDA = temp & 0x01 → 0 → SDA=0（发送bit3）
    temp >>= 1  → temp = 0000 1000
    SCK = 1

第5轮：
    SCK = 0
    SDA = temp & 0x01 → 0 → SDA=0（发送bit4）
    temp >>= 1  → temp = 0000 0100
    SCK = 1

第6轮：
    SCK = 0
    SDA = temp & 0x01 → 0 → SDA=0（发送bit5）
    temp >>= 1  → temp = 0000 0010
    SCK = 1

第7轮：
    SCK = 0
    SDA = temp & 0x01 → 0 → SDA=0（发送bit6）
    temp >>= 1  → temp = 0000 0001
    SCK = 1

第8轮：
    SCK = 0
    SDA = temp & 0x01 → 1 → SDA=1（发送bit7）
    temp >>= 1  → temp = 0000 0000
    SCK = 1

发送顺序：0→0→1→0→0→0→0→1（从低位到高位）
即：bit0, bit1, bit2, bit3, bit4, bit5, bit6, bit7

最终线上的数据：0x84（但按位顺序是 0010 0001 的逆序发送）
```

## 四、函数二：Write_Ds1302_Byte - 单寄存器写入（地址+数据）

### 代码实现

```c
void Write_Ds1302_Byte(unsigned char address, unsigned char dat)
{
    RST = 0;
    _nop_(); // RST拉低复位，_nop_()插入1个机器周期延时确保稳定
    SCK = 0;
    _nop_(); // SCK拉低，确保时钟线初始为低电平
    RST = 1;
    _nop_();               // RST拉高，启动数据传输（CE使能）
    Write_Ds1302(address); // 发送命令字节（地址+读写标志）
    Write_Ds1302(dat);     // 发送数据字节
    RST = 0;               // RST拉低，结束传输
}
```

```
    SCK = 0;    _nop_();  // SCK拉低，时钟线初始为低

步骤2：启动传输
    RST = 1;    _nop_();  // RST拉高，DS1302使能，开始通信

步骤3：发送地址 0x80
    调用 Write_Ds1302(0x80)

    第2轮：SCK=0, SDA=0, temp=0001 0101, SCK=1  → 发bit1=0
    第3轮：SCK=0, SDA=1, temp=0000 1010, SCK=1  → 发bit2=1
```

## 五、函数三：Read_Ds1302_Byte - 单寄存器读取

### 代码实现

```c
unsigned char Read_Ds1302_Byte(unsigned char address)
{
    unsigned char i, temp = 0x00;
  
    RST = 0;    _nop_();
    SCK = 0;    _nop_();
    RST = 1;    _nop_();
  
    Write_Ds1302(address);    // 发送要读取的地址
  
    for (i=0; i<8; i++)       // 读取8位数据
    {
        SCK = 0;              // 时钟拉低
        temp >>= 1;           // temp右移，腾出最高位
        if(SDA)               // 读取IO线上的数据
            temp |= 0x80;     // 如果高电平，最高位置1
        SCK = 1;              // 时钟拉高
    }
  
    RST = 0;
    return temp;              // 返回读取的字节
}
```

### 执行过程示例：从秒寄存器读取数据（假设DS1302返回 `0x35`，即35秒）

```
调用：Read_Ds1302_Byte(0x81)  // 0x81是秒寄存器的读地址

步骤1：初始化通信
    RST = 0;    _nop_();  // RST拉低，总线复位
    SCK = 0;    _nop_();  // SCK拉低，时钟线初始化为低

步骤2：启动传输
    RST = 1;    _nop_();  // RST拉高，启动传输

步骤3：发送读地址 0x81
    调用 Write_Ds1302(0x81)
    
    0x81 = 1000 0001（二进制）
    
    第1轮：SCK=0, SDA=1, temp=0100 0000, SCK=1  → 发bit0=1（读标志）
    第2轮：SCK=0, SDA=0, temp=0010 0000, SCK=1  → 发bit1=0
    第3轮：SCK=0, SDA=0, temp=0001 0000, SCK=1  → 发bit2=0
    第4轮：SCK=0, SDA=0, temp=0000 1000, SCK=1  → 发bit3=0
    第5轮：SCK=0, SDA=0, temp=0000 0100, SCK=1  → 发bit4=0
    第6轮：SCK=0, SDA=0, temp=0000 0010, SCK=1  → 发bit5=0
    第7轮：SCK=0, SDA=0, temp=0000 0001, SCK=1  → 发bit6=0
    第8轮：SCK=0, SDA=1, temp=0000 0000, SCK=1  → 发bit7=1
    
    发送完成：0x81（秒寄存器读地址，bit0=1表示读操作）

步骤4：读取数据（DS1302返回 0x35 = 0011 0101）
    temp = 0x00（初始值）
    
    第1轮（读bit0=1）：
        SCK = 0                    // 时钟拉低，DS1302将bit0放到SDA
        temp >>= 1 → 0000 0000     // temp右移（此时还是0）
        SDA = 1 → temp |= 0x80 → 1000 0000  // 最高位置1
        SCK = 1                    // 时钟拉高，采样完成
        
    第2轮（读bit1=0）：
        SCK = 0
        temp >>= 1 → 0100 0000     // 右移
        SDA = 0 → temp不变 → 0100 0000
        SCK = 1
        
    第3轮（读bit2=1）：
        SCK = 0
        temp >>= 1 → 0010 0000     // 右移
        SDA = 1 → temp |= 0x80 → 1010 0000  // 最高位置1
        SCK = 1
        
    第4轮（读bit3=0）：
        SCK = 0
        temp >>= 1 → 0101 0000     // 右移
        SDA = 0 → temp不变 → 0101 0000
        SCK = 1
        
    第5轮（读bit4=1）：
        SCK = 0
        temp >>= 1 → 0010 1000     // 右移
        SDA = 1 → temp |= 0x80 → 1010 1000  // 最高位置1
        SCK = 1
        
    第6轮（读bit5=1）：
        SCK = 0
        temp >>= 1 → 0101 0100     // 右移
        SDA = 1 → temp |= 0x80 → 1101 0100  // 最高位置1
        SCK = 1
        
    第7轮（读bit6=0）：
        SCK = 0
        temp >>= 1 → 0110 1010     // 右移
        SDA = 0 → temp不变 → 0110 1010
        SCK = 1
        
    第8轮（读bit7=0）：
        SCK = 0
        temp >>= 1 → 0011 0101     // 右移 ← 这就是0x35！
        SDA = 0 → temp不变 → 0011 0101
        SCK = 1

步骤5：结束传输并复位总线
    RST = 0;    _nop_();  // 结束传输
    SCK = 0;    _nop_();  // 时钟线复位
    SCK = 1;    _nop_();
    SDA = 0;    _nop_();  // 数据线复位
    SDA = 1;    _nop_();

返回值：temp = 0x35（BCD码，表示35秒）

数据接收过程图解：
初始：temp = 0000 0000

第1轮后：1000 0000  ← 收到bit0=1，放到最高位
第2轮后：0100 0000  ← 右移，收到bit1=0
第3轮后：1010 0000  ← 右移，收到bit2=1，最高位置1
第4轮后：0101 0000  ← 右移，收到bit3=0
第5轮后：1010 1000  ← 右移，收到bit4=1，最高位置1
第6轮后：1101 0100  ← 右移，收到bit5=1，最高位置1
第7轮后：0110 1010  ← 右移，收到bit6=0
第8轮后：0011 0101  ← 右移，收到bit7=0，得到 0x35
```

## 六、完整写入操作执行流程（批量写入7个时间寄存器）

### 代码实现

```c
void DS1302_Write_Time()
{
    Write_Ds1302_Byte(0x8e, 0x00); // 关闭写保护（WP=0）
    for (i = 0; i < 7; i++)         // 循环7次
    {
        Write_Ds1302_Byte(DS1302_Write_adr[i], DS1302_Time_Reset[i]);
    }
    Write_Ds1302_Byte(0x8e, 0x80); // 开启写保护（WP=1）
}
```

### 执行过程示例

```
初始数据：
DS1302_Write_adr[7] = {0x80, 0x82, 0x84, 0x86, 0x88, 0x8A, 0x8C}

DS1302_Time_Reset[7] = {0x55, 0x05, 0x23, 0x09, 0x02, 0x07, 0x25}
                        // 55秒 05分 23时 09日 02月 周日  25年

完整执行流程：

【步骤1】关闭写保护
    Write_Ds1302_Byte(0x8E, 0x00)
    ├── RST=0, SCK=0          // 初始化
    ├── RST=1                 // 启动传输
    ├── Write_Ds1302(0x8E)    // 发送控制寄存器地址
    │   └── 发送 1000 0110（0x8E的二进制，LSB先发）
    ├── Write_Ds1302(0x00)    // 发送数据0x00（WP=0，关闭写保护）

【步骤2】循环写入7个时间寄存器

    第1次循环（i=0）：写入秒
    Write_Ds1302_Byte(0x80, 0x55)
    ├── RST=0, SCK=0          // 初始化
    ├── RST=1                 // 启动传输
    ├── Write_Ds1302(0x80)    // 发送秒寄存器地址
    ├── Write_Ds1302(0x55)    // 发送55秒（BCD码）

    第2次循环（i=1）：写入分
    Write_Ds1302_Byte(0x82, 0x05)
    ├── RST=0, SCK=0
    ├── RST=1
    ├── Write_Ds1302(0x82)    // 发送分寄存器地址
    ├── Write_Ds1302(0x05)    // 发送05分（BCD码）
    └── RST=0
    
    第3次循环（i=2）：写入时
    Write_Ds1302_Byte(0x84, 0x23)
    ├── 发送地址 0x84
    ├── 发送数据 0x23（23时）
    
    第4次循环（i=3）：写入日
    Write_Ds1302_Byte(0x86, 0x09)
    ├── 发送地址 0x86
    ├── 发送数据 0x09（09日）
    
    第5次循环（i=4）：写入月
    Write_Ds1302_Byte(0x88, 0x02)
    ├── 发送地址 0x88
    ├── 发送数据 0x02（02月）
    
    第6次循环（i=5）：写入星期
    Write_Ds1302_Byte(0x8A, 0x07)
    ├── 发送地址 0x8A
    ├── 发送数据 0x07（周日）
    
    第7次循环（i=6）：写入年
    Write_Ds1302_Byte(0x8C, 0x25)
    ├── 发送地址 0x8C
    └── 发送数据 0x25（25年）

【步骤3】开启写保护

    ├── RST=0, SCK=0          // 初始化
    ├── RST=1                 // 启动传输
    ├── Write_Ds1302(0x8E)    // 发送控制寄存器地址
    ├── Write_Ds1302(0x80)    // 发送数据0x80（WP=1，开启写保护）
    │   └── 0x80 = 1000 0000，bit7=1表示WP置位
    └── RST=0                 // 结束传输

总通信次数：9次（1次关保护 + 7次写时间 + 1次开保护）
每次通信包含：RST起始 + 8位地址 + 8位数据 + RST结束
```

## 七、完整读取操作执行流程（批量读取7个时间寄存器）

### 代码实现

```c
void DS1302_Read_Time()
{
    // 第一步：批量读取7个寄存器的BCD时间数据
    for (i = 0; i < 7; i++)
    {
        DS1302_Time_Reset[i] = Read_Ds1302_Byte(DS1302_Read_adr[i]);
    }

    // 第二步：BCD转十进制
    for (i = 0; i < 7; i++)
    {
        Time10[i] = DS1302_Time_Reset[i] / 16 * 10 + DS1302_Time_Reset[i] % 16;
    }
}
```

### 执行过程示例

```
寄存器读地址：
DS1302_Read_adr[7] = {0x81, 0x83, 0x85, 0x87, 0x89, 0x8B, 0x8D}
                      // 秒   分   时   日   月   星期  年

假设DS1302中存储的时间：
2025年2月9日 23:05:55 星期日

【步骤1】批量读取7个寄存器

    第1次循环（i=0）：读取秒
    DS1302_Time_Reset[0] = Read_Ds1302_Byte(0x81)
    ├── RST=0, SCK=0          // 初始化
    ├── RST=1                 // 启动传输
    ├── Write_Ds1302(0x81)    // 发送秒寄存器读地址
    ├── for循环读取8位       // DS1302返回 0x55
    │   └── temp经过8轮移位组装：最终 = 0x55
    ├── RST=0                 // 结束传输
    ├── SCK/SDA复位序列
    └── 返回 0x55
    
    第2次循环（i=1）：读取分
    DS1302_Time_Reset[1] = Read_Ds1302_Byte(0x83)

    第3次循环（i=2）：读取时
    DS1302_Time_Reset[2] = Read_Ds1302_Byte(0x85)
    └── 返回 0x23
    
    第4次循环（i=3）：读取日
    DS1302_Time_Reset[3] = Read_Ds1302_Byte(0x87)
    └── 返回 0x09
```

|------|------|------|---------------|
| **Write_Ds1302** | 底层单字节发送 | `temp`（8位数据） | 被上层函数调用，一般不直接使用 |
| **Write_Ds1302_Byte** | 单寄存器写入 | `address`（地址）+ `dat`（数据） | 写时间、开关保护等 |
| **Read_Ds1302_Byte** | 单寄存器读取 | `address`（读地址） | 读取时间寄存器 |

**调用关系**：
```
DS1302_Write_Time()          DS1302_Read_Time()
        │                            │
        ▼                            ▼
Write_Ds1302_Byte()          Read_Ds1302_Byte()
        │                            │
        ▼                            ▼
Write_Ds1302()               Write_Ds1302() + 读取循环
```
