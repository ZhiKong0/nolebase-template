# STM32

## 一、RCC（复位与时钟控制器）

### 核心认识

- **RCC 是所有外设的"心脏"** - 没有时钟，外设无法工作
- 负责系统时钟配置（72MHz）和各外设时钟使能

### 关键公式

```
定时器时钟 = 系统时钟 / (预分频+1)

实际应用：
- APB1总线 = 36MHz，但TIM2~7实际时钟 = 72MHz（×2倍频）
- APB2总线 = 72MHz，TIM1/8 = 72MHz
```

### 配置流程记忆

```
1. 启用HSE（外部8MHz晶振）
2. 配置Flash等待周期（>48MHz需2个wait states）
3. 配置AHB/APB分频器
4. 配置PLL（8MHz × 9 = 72MHz）
5. 切换到PLL作为系统时钟
6. 使能各外设时钟
```

### 易错点 ⚠️

- **APB1不能超过36MHz！**
- **使用任何外设前必须使能时钟**
- **GPIO重映射需要AFIO时钟**

## 二、NVIC（嵌套向量中断控制器）

- [[NVIC详解]]

### 核心认识

- **NVIC是"总指挥"** - 管理所有中断的优先级和响应
- 是Cortex-M内核外设，所有STM32都有

### 两种优先级

| 类型                 | 功能                 | 类比             |
| :------------------- | :------------------- | :--------------- |
| **抢占优先级** | 可打断正在执行的中断 | 急诊病人插队     |
| **响应优先级** | 仅影响排队顺序       | 普通病人优先叫号 |

### 优先级分组（最常用Group 2）

```
Group 2: 2位抢占 + 2位响应
- 抢占优先级范围：0~3（0最高）
- 响应优先级范围：0~3
```

### 中断向量表作用

```
EXTI检测变化 ──► 发送请求给NVIC ──► NVIC查向量表 ──► 跳转到ISR执行
```

## 三、EXTI（外部中断控制器）

- [[EXTI详解]]

### 核心认识

- **EXTI是"侦察兵"** - 检测GPIO电平变化
- 是芯片外设，不是NVIC的一部分

### 关键特性

| 项目     | 内容                                       |
| :------- | :----------------------------------------- |
| 外部线   | 16条（EXTI0~15对应Pin0~15）               |
| 触发方式 | 上升沿/下降沿/双边沿/软件触发              |
| 输出模式 | 中断模式（执行ISR）/ 事件模式（仅唤醒MCU） |

### GPIO到EXTI的映射

```
PA0/PB0/PC0... ──► AFIO选择器 ──► EXTI0 ──► NVIC
```

**注意：同一时刻只能有一个端口的PinX映射到EXTIx**

### 配置流程

```
1. 使能GPIO时钟 + AFIO时钟（重要！）
2. 配置GPIO模式（输入上拉/浮空）
3. GPIO_EXTILineConfig() 映射引脚到EXTI线
4. EXTI_Init() 配置触发方式
5. NVIC_Init() 配置中断优先级
```

### 按键消抖

- **简单方式**：ISR中延时20ms后确认按键状态
- **推荐方式**：ISR标记事件，定时器中断中执行消抖

## 四、TIM（定时器）

- [[TIM详解]]

### 核心认识

- **TIM是"计时员"** - 精确计时和产生波形
- 可以产生中断，也可以不产生中断

### 定时器分类

| 类型 | 定时器    | 特点                       |
| :--- | :-------- | :------------------------- |
| 高级 | TIM1/TIM8 | 带互补输出和死区，用于电机 |
| 通用 | TIM2~5    | 功能最全，最常用           |
| 基本 | TIM6/7    | 仅定时，用于DAC触发        |

### 时基计算公式

```
定时频率 = 时钟频率 / (PSC+1) / (ARR+1)

例：1ms定时（时钟72MHz）
PSC = 72-1, ARR = 1000-1
72MHz / 72 / 1000 = 1kHz = 1ms
```

### 中断 vs 非中断模式

| 模式       | 应用场景            |
| :--------- | :------------------ |
| 中断模式   | 系统节拍、定时任务  |
| 非中断模式 | 纯PWM输出、简单延时 |

## 五、各组件关系总结

```
┌──────────────────────────────────────────────────────────┐
│                    STM32中断系统架构                       │
├──────────────────────────────────────────────────────────┤
│                                                           │
│  ┌─────────┐    ┌─────────┐    ┌─────────┐    ┌────────┐ │
│  │  GPIO   │───►│  EXTI   │───►│  NVIC   │───►│  CPU   │ │
│  │  (输入) │    │ (检测)  │    │ (管理)  │    │ (执行) │ │
│  └─────────┘    └─────────┘    └────┬────┘    └────────┘ │
│                                     │                     │
│                    ┌─────────┐      │                     │
│                    │  TIM    │──────┘                     │
│                    │ (定时)  │                            │
│                    └────┬────┘                            │
│                         │                                 │
│                    ┌────┴────┐                            │
│                    │ RCC时钟 │◄─── 所有外设的心脏          │
│                    └─────────┘                            │
│                                                           │
│  中断向量表：存储各ISR函数地址，NVIC通过它知道跳转位置       │
│                                                           │
└──────────────────────────────────────────────────────────┘
```

### 数据流向

| 路径            | 说明                    |
| :-------------- | :---------------------- |
| GPIO → EXTI    | 电平变化信号            |
| EXTI → NVIC    | 中断请求（IRQ Line）    |
| NVIC → CPU     | 中断响应 + 查向量表跳转 |
| RCC → 所有外设 | 时钟信号                |

## 六、实际应用场景

### 场景1：按键控制LED

```
PA0（按键）──► EXTI0 ──► NVIC（优先级2）──► ISR中翻转LED
```

### 场景2：定时采集传感器

```
TIM2（1ms中断）──► NVIC（优先级1）──► ISR中读取ADC值
```

### 场景3：电机PWM控制

```
TIM1（高级定时器）──► PWM输出 ──► 驱动电机（无需中断）
```

## 七、今日收获

### 理解的关键点

1. ✅ RCC是时钟源头，所有外设都依赖它
2. ✅ NVIC是中断管理者，EXTI/TIM等是中断产生者
3. ✅ 中断向量表存的是函数地址，不是连接组件
4. ✅ APB1定时器时钟 = 36MHz × 2 = 72MHz
5. ✅ 按键必须消抖，EXTI和TIM都可以配合NVIC使用

### 记忆口诀

> **"RCC供时钟，EXTI检变化，NVIC管响应，TIM做计时"**
> **"APB1乘二算时钟，PSC分频ARR计，抢占嵌套响应排"**
