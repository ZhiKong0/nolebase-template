## 组件

### NVIC（嵌套向量中断控制器）

**NVIC**（Nested Vectored Interrupt Controller）是 Cortex-M 内核中的**中断管理单元**，负责管理所有中断的优先级、使能/禁用和响应。

| 特性   | 说明                                       |
| ------ | ------------------------------------------ |
| 功能   | 中断优先级裁决、中断嵌套管理、中断使能控制 |
| 优先级 | 16级可编程优先级（STM32F1使用高4位）       |
| 分组   | 支持抢占优先级和响应优先级分组             |
| 位置   | 内核外设，位于 Cortex-M 处理器内部         |

**作用**：当多个外设同时请求中断时，NVIC 根据优先级决定哪个中断先被 CPU 处理，支持高优先级中断**抢占**（打断）低优先级中断。

---

### MPU（内存保护单元）

**MPU**（Memory Protection Unit）是 Cortex-M 系列中的**内存访问控制单元**，用于限制程序对不同内存区域的访问权限。

| 特性 | 说明                                           |
| ---- | ---------------------------------------------- |
| 功能 | 定义内存区域的访问权限（读/写/执行）           |
| 区域 | 最多支持 8 个独立内存区域                      |
| 用途 | 防止程序越界访问、实现任务隔离、增强系统安全性 |
| 位置 | 内核外设，可选组件（部分型号可能没有）         |

**作用**：在多任务系统或安全敏感应用中，MPU 可以防止一个任务意外破坏其他任务的内存空间，提高系统稳定性。

---

### SysTick（系统定时器）

**SysTick** 是 Cortex-M 内核内置的**24位递减计数定时器**，专为操作系统或需要周期性中断的应用设计。

| 特性   | 说明                                              |
| ------ | ------------------------------------------------- |
| 功能   | 提供固定时间间隔的定时中断                        |
| 时钟源 | 可选内核时钟 (HCLK) 或 HCLK/8                     |
| 用途   | 实现延时函数、操作系统心跳 (tick)、周期性任务调度 |
| 位置   | 内核外设，所有 Cortex-M 处理器都有                |

**作用**：SysTick 是嵌入式系统中实现**精确延时**和**任务调度**的基础，例如 HAL 库的 `HAL_Delay()` 函数就是基于 SysTick 实现的。

#### SysTick vs TIM（定时器区别）

| 对比项 | SysTick | TIM（通用/高级定时器） |
| ------ | ------- | ---------------------- |
| **位置** | 内核外设（所有 Cortex-M 都有） | 片内外设（STM32特有） |
| **位数** | 24位递减计数器 | 16位（TIM2/5为32位）递增/递减 |
| **时钟源** | HCLK 或 HCLK/8 | 多种（内部/外部/其他定时器） |
| **通道数** | 1个 | 多个（TIM1~TIM8等） |
| **功能** | 简单定时、延时 | PWM、输入捕获、编码器、死区控制等 |
| **用途** | 系统节拍、任务调度、延时 | 电机控制、波形生成、精准计时 |
| **代码依赖** | 可直接寄存器操作 | 需配置 GPIO、时钟、中断等 |

**简单记忆**：
- **SysTick** = 系统"心跳"，做简单的周期性事情（如延时1ms）
- **TIM** = 功能强大的"瑞士军刀"，做复杂的定时任务（如PWM控制电机转速）

#### 深入理解：芯片的"洋葱结构"

**重要澄清**：两者都是 STM32 **自带的**，但来自不同层级。

```
┌─────────────────────────────────────────┐
│           STM32F103C8T6 芯片             │
│  ┌─────────────────────────────────┐    │
│  │      Cortex-M3 内核 (ARM设计)  │    │
│  │  ┌─────────────────────────┐   │    │
│  │  │  • NVIC (中断控制器)    │   │    │
│  │  │  • SysTick (系统定时器) │ ← 内核外设 │
│  │  │  • MPU (内存保护)       │   │    │
│  │  │  • SCB (系统控制)       │   │    │
│  │  └─────────────────────────┘   │    │
│  └─────────────────────────────────┘    │
│                                         │
│  ┌─────────────────────────────────┐   │
│  │    ST公司添加的片内外设          │   │
│  │  ┌─────────────────────────┐   │   │
│  │  │  • TIM1-TIM8 (定时器)    │   │   │
│  │  │  • GPIOA-G (通用IO)      │   │   │
│  │  │  • USART1-3 (串口)       │   │   │
│  │  │  • ADC1-2 (模数转换)     │   │   │
│  │  │  • SPI/I2C/CAN/USB...    │   │   │
│  │  └─────────────────────────┘   │   │
│  └─────────────────────────────────┘   │
└─────────────────────────────────────────┘
```

**关键区别**：

| 对比项 | SysTick | TIM |
| ------ | ------- | --- |
| **设计者** | ARM 公司 | ST 公司 |
| **位置** | Cortex-M 内核**内部** | Cortex-M 内核**外部** |
| **通用性** | 所有 Cortex-M 芯片都有 | 只有 STM32 有（其他厂商有别的定时器） |

**举个例子**：

| 芯片 | 内核 | SysTick | 定时器 |
|------|------|---------|--------|
| STM32F103 | Cortex-M3 | ✅ 有 | ✅ TIM1-TIM8 |
| GD32F103（兆易创新） | Cortex-M3 | ✅ 有 | ✅ 自己的定时器 |
| NXP LPC1768 | Cortex-M3 | ✅ 有 | ✅ 自己的定时器 |

**发现了吗？**
- 只要是 **Cortex-M3**，就一定有 **SysTick**（ARM 强制要求）
- 但 **TIM** 只有 ST 的芯片才有，其他公司用 Cortex-M3 做芯片时会设计自己的定时器

**为什么这样设计？**

- **ARM**："每个嵌入式系统都需要基本的定时中断来做延时和调度，我给你们内置一个最简单的 SysTick。"
- **ST**："只做延时太简单了，我还要控制电机、生成 PWM、测量脉冲... 我要设计功能强大的 TIM 定时器。"

**简单类比**：

```
SysTick = 内裤（内核自带，必须穿）
TIM = 外套（厂商自己设计，功能多样）
```

**使用选择**：
- 只要**延时** → SysTick（简单方便，`HAL_Delay()` 就用它）
- 要**PWM/输入捕获/编码器** → TIM（功能强大）

---

**三者关系**：

- **NVIC** 负责中断管理
- **MPU** 负责内存保护
- **SysTick** 负责定时中断

它们都是 Cortex-M 内核的**核心外设**，位于处理器内部，与 GPIO、TIM 等片内外设不同。

## STM32 外设汇总（STM32F103 为例）

### 一、Cortex-M3 内核外设（ARM设计）

| 外设 | 全称 | 功能描述 |
|------|------|----------|
| **NVIC** | Nested Vectored Interrupt Controller | 嵌套向量中断控制器，管理中断优先级和响应 |
| **SysTick** | System Tick Timer | 系统定时器，提供延时和任务调度 |
| **MPU** | Memory Protection Unit | 内存保护单元，限制内存访问权限（可选） |
| **SCB** | System Control Block | 系统控制块，控制电源、异常、时钟等 |
| **FPB** | Flash Patch and Breakpoint | 闪存补丁和断点单元，调试使用 |
| **DWT** | Data Watchpoint and Trace | 数据观察点和跟踪单元，调试使用 |
| **ITM** | Instrumentation Trace Macrocell | 仪器跟踪宏单元，调试使用 |

> **注意**：内核外设是 ARM 设计的，所有 Cortex-M3 芯片都有，与厂商无关。

---

### 二、ST 片内外设（STM32F103Cx 系列）

#### 1. 系统与时钟

| 外设 | 功能描述 | 数量 |
|------|----------|------|
| **RCC** | Reset and Clock Control，复位和时钟控制 | 1 |
| **PWR** | Power Control，电源控制（低功耗管理） | 1 |
| **BKP** | Backup Registers，备份寄存器 | 1 |
| **CRC** | CRC Calculation Unit，CRC校验计算单元 | 1 |

#### 2. GPIO 与中断

| 外设 | 功能描述 | 数量 |
|------|----------|------|
| **GPIOA~G** | General Purpose Input/Output，通用输入输出 | 7组 |
| **AFIO** | Alternate Function I/O，复用功能IO（引脚重映射） | 1 |
| **EXTI** | External Interrupt/Event Controller，外部中断/事件控制器 | 1 |

#### 3. 定时器类

| 外设 | 类型 | 位数 | 功能特点 |
|------|------|------|----------|
| **TIM1** | 高级定时器 | 16位 | 带死区控制、刹车输入，用于电机控制 |
| **TIM8** | 高级定时器 | 16位 | 同上（大容量产品才有） |
| **TIM2** | 通用定时器 | **32位** | 最通用的定时器 |
| **TIM5** | 通用定时器 | **32位** | 同上（大容量产品才有） |
| **TIM3, TIM4** | 通用定时器 | 16位 | 通用定时功能 |
| **TIM6** | 基本定时器 | 16位 | 仅定时/触发DAC |
| **TIM7** | 基本定时器 | 16位 | 同上（大容量产品才有） |

#### 4. 通信接口

| 外设 | 全称 | 数量 | 功能描述 |
|------|------|------|----------|
| **USART1~3** | Universal Synchronous Asynchronous Receiver Transmitter | 3 | 串口通信（USART1 最高72M，其他36M） |
| **SPI1, SPI2** | Serial Peripheral Interface | 2 | 高速同步串行通信 |
| **I2C1, I2C2** | Inter-Integrated Circuit | 2 | 两线串行总线通信 |
| **USB** | Universal Serial Bus | 1 | 全速 USB 2.0 设备接口 |
| **CAN** | Controller Area Network | 1 | CAN 总线通信（ bxCAN ） |
| **SDIO** | SD/SDIO/MMC Interface | 1 | SD 卡接口（大容量产品才有） |
| **ETH** | Ethernet MAC | 1 | 以太网（互联型才有） |

#### 5. 模拟外设

| 外设 | 全称 | 数量 | 功能描述 |
|------|------|------|----------|
| **ADC1, ADC2** | Analog-to-Digital Converter | 2 | 12位模数转换器，最多16通道 |
| **DAC1, DAC2** | Digital-to-Analog Converter | 2 | 12位数模转换器（互联型才有） |
| **TEMP SENSOR** | Temperature Sensor | 1 | 内部温度传感器，接ADC1通道16 |
| **VREFINT** | Internal Reference Voltage | 1 | 内部参考电压，接ADC1通道17 |

#### 6. 看门狗与 RTC

| 外设 | 全称 | 功能描述 |
|------|------|----------|
| **WWDG** | Window Watchdog | 窗口看门狗，必须在时间窗口内喂狗 |
| **IWDG** | Independent Watchdog | 独立看门狗，独立时钟，用于检测程序跑飞 |
| **RTC** | Real-Time Clock | 实时时钟，由备份域供电，掉电保持 |

#### 7. DMA 与存储

| 外设 | 全称 | 数量 | 功能描述 |
|------|------|------|----------|
| **DMA1** | Direct Memory Access | 1 | 7个通道，内存与外设间数据搬运 |
| **DMA2** | Direct Memory Access | 1 | 5个通道（大容量产品才有） |
| **FLASH** | Flash Memory Interface | 1 | 闪存接口，用于读写内部Flash |
| **FSMC** | Flexible Static Memory Controller | 1 | 灵活的静态存储器控制器（互联型/大容量才有） |


**记忆口诀**：
- 内核外设：`N`VIC + `S`ysTick + `M`PU + `S`CB = **"NSMS"**
- 片内外设：时钟 + GPIO + 定时器 + 通信 + 模拟 + 看门狗 + RTC + DMA + 存储
