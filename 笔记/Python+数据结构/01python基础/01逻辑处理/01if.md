## 一、基础语法

### 1.1 三种基本形式

```python
# 形式1: 只有 if
if 条件:
    代码块

# 形式2: if-else
if 条件:
    代码块A
else:
    代码块B

# 形式3: if-elif-else（多分支）
if 条件1:
    代码块1
elif 条件2:
    代码块2
elif 条件3:
    代码块3
else:
    代码块N
```

### 1.2 关键规则

| 规则 | 说明 | 示例 |
|:-----|:-----|:-----|
**冒号** | 条件后必须有 `:` | `if x > 0:`
**缩进** | 代码块统一缩进（通常4空格） | 不能混用Tab和空格
**缩进级别** | 同一代码块必须同级 | `if` 和 `else` 对齐

## 二、条件表达式

### 2.1 比较运算符

| 运算符 | 含义 | 示例 | 结果 |
|:------:|:-----|:-----|:----:|
| `==` | 等于 | `5 == 5` | `True` |
| `!=` | 不等于 | `5 != 3` | `True` |
| `>` | 大于 | `5 > 3` | `True` |
| `<` | 小于 | `5 < 3` | `False` |
| `>=` | 大于等于 | `5 >= 5` | `True` |
| `<=` | 小于等于 | `5 <= 3` | `False` |

### 2.2 逻辑运算符

| 运算符 | 含义 | 示例 | 说明 |
|:------:|:-----|:-----|:-----|
| `and` | 与 | `x > 0 and x < 10` | 两者都为真才为真 |
| `or` | 或 | `x < 0 or x > 10` | 任一真即为真 |
| `not` | 非 | `not x == 0` | 取反 |

```python
# 逻辑运算示例
age = 25
income = 5000

# and: 同时满足
if age >= 18 and income >= 3000:
    print("符合申请条件")

# or: 满足其一
if age < 18 or age > 60:
    print("享受优惠票价")

# not: 取反
if not (age < 18):
    print("已成年")
```

### 2.3 真值判断

```python
# 以下值在条件中视为 False:
# False, 0, 0.0, "" (空字符串), [] (空列表), {} (空字典), None

# 示例
name = ""
if name:
    print("有名字")
else:
    print("名字为空")  # 输出这行

# 判断列表非空
data = []
if data:
    process(data)
else:
    print("暂无数据")
```

## 三、嵌套与组合

### 3.1 if 嵌套

```python
# 嵌套结构
score = 85

if score >= 60:
    print("及格")
    if score >= 90:
        print("优秀")
    elif score >= 80:
        print("良好")  # 输出这行
    else:
        print("中等")
else:
    print("不及格")
```

### 3.2 多条件组合

```python
# 复杂条件判断
user_type = "member"
purchase_amount = 250
is_new_user = False

# 复杂条件组合
if (user_type == "member" and purchase_amount >= 200) or \
   (is_new_user and purchase_amount >= 100):
    print("享受折扣")
else:
    print("不享受折扣")

# 更清晰的多分支写法
if is_new_user:
    threshold = 100
else:
    threshold = 200

if user_type == "member" and purchase_amount >= threshold:
    print("享受折扣")
```

## 四、三元表达式

### 4.1 基本语法

```python
# 语法: 值1 if 条件 else 值2
# 条件为真返回值1，否则返回值2

# 示例1: 取最大值
a, b = 10, 20
max_val = a if a > b else b
print(max_val)  # 20

# 示例2: 判断奇偶
num = 15
result = "奇数" if num % 2 == 1 else "偶数"
print(result)  # 奇数

# 示例3: 简化赋值
score = 75
grade = "及格" if score >= 60 else "不及格"
```

### 4.2 与 if-else 对比

| 场景 | 传统写法 | 三元表达式 |
|:-----|:---------|:-----------|
| 简单赋值 | 4行 | 1行 |
| 复杂逻辑 | 推荐 | 不推荐（可读性差） |

```python
# 不推荐：嵌套三元（可读性差）
result = "A" if score >= 90 else ("B" if score >= 80 else ("C" if score >= 60 else "D"))

# 推荐：多分支更清晰
if score >= 90:
    result = "A"
elif score >= 80:
    result = "B"
elif score >= 60:
    result = "C"
else:
    result = "D"
```

## 五、常见错误

### 5.1 语法错误

```python
# ❌ 错误1: 忘记冒号
if x > 0
    print("正数")

# ✅ 正确
if x > 0:
    print("正数")

# ❌ 错误2: 缩进不一致
if x > 0:
    print("正数")
     print("大于0")  # 缩进不一致

# ✅ 正确
if x > 0:
    print("正数")
    print("大于0")

# ❌ 错误3: 混用空格和Tab
if x > 0:
    print("正数")  # Tab缩进
    print("大于0")  # 4空格缩进
```

### 5.2 逻辑错误

```python
# ❌ 错误4: 使用 = 而不是 ==
if x = 5:  # 这是赋值，不是比较
    print("x等于5")

# ✅ 正确
if x == 5:
    print("x等于5")

# ❌ 错误5: 多条件写错
if x > 0 and < 10:  # 缺少比较对象
    print("0到10之间")

# ✅ 正确
if x > 0 and x < 10:
    print("0到10之间")

# 或更简洁的写法
if 0 < x < 10:  # Python支持链式比较
    print("0到10之间")

# ❌ 错误6: 判断相等用 is（对象身份）
if x is 5:  # 不要这样用！
    pass

# ✅ 正确
if x == 5:  # 值比较
    pass
```

## 六、实战示例

### 6.1 成绩评级

```python
def grade_score(score):
    """根据分数返回等级"""
    if not (0 <= score <= 100):
        return "无效分数"
    
    if score >= 90:
        return "A (优秀)"
    elif score >= 80:
        return "B (良好)"
    elif score >= 70:
        return "C (中等)"
    elif score >= 60:
        return "D (及格)"
    else:
        return "F (不及格)"

# 测试
print(grade_score(95))  # A (优秀)
print(grade_score(82))  # B (良好)
print(grade_score(55))  # F (不及格)
```

### 6.2 用户权限判断

```python
def check_access(user_role, is_logged_in):
    """检查用户访问权限"""
    if not is_logged_in:
        return "请先登录"
    
    if user_role == "admin":
        return "完整管理权限"
    elif user_role == "editor":
        return "编辑权限"
    elif user_role == "user":
        return "普通用户权限"
    else:
        return "未知角色"

# 测试
print(check_access("admin", True))   # 完整管理权限
print(check_access("user", False))   # 请先登录
```

### 6.3 日期判断（闰年）

```python
def is_leap_year(year):
    """判断是否为闰年"""
    if (year % 4 == 0 and year % 100 != 0) or (year % 400 == 0):
        return True
    return False

# 简化版
# return (year % 4 == 0 and year % 100 != 0) or (year % 400 == 0)

# 测试
print(is_leap_year(2020))  # True
print(is_leap_year(1900))  # False (能被100整除但不能被400整除)
print(is_leap_year(2000))  # True (能被400整除)
```

## 知识点速查表

| 语法 | 用途 | 示例 |
|:-----|:-----|:-----|
| `if` | 单条件判断 | `if x > 0:` |
| `if-else` | 二选一 | `if x > 0: ... else: ...` |
| `if-elif-else` | 多分支 | `if ... elif ... else` |
| `==` `!=` | 等于/不等于 | `x == 5` |
| `>` `<` `>=` `<=` | 大小比较 | `x >= 10` |
| `and` `or` `not` | 逻辑运算 | `x > 0 and x < 10` |
| `A if C else B` | 三元表达式 | `"正" if x>0 else "负"` |
