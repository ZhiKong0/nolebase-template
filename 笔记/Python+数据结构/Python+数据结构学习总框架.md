## 一、学习路线图总览

### 1.1 整体架构

```
┌─────────────────────────────────────────────────────────┐
│                   Python + 数据结构                      │
│                    完整学习体系                          │
├─────────────────────────────────────────────────────────┤

│  ├── 语法基础 → 数据类型 → 流程控制 → 函数              │

│  ├── 基础结构: 数组 → 链表 → 栈 → 队列                  │
│  ├── 树形结构: 二叉树 → BST → 堆 → 平衡树               │
│  ├── 图结构: 邻接表/矩阵 → BFS/DFS → 最短路径           │
```

### 2.3 核心知识点速查

| 概念 | 关键内容 | 数据结构关联 |
|:-----|:---------|:------------|
| 列表(list) | 动态数组，O(1)访问，O(n)插入删除 | 线性表/动态数组实现 |
| 字典(dict) | 哈希表实现，O(1)平均查找 | 哈希表原理基础 |
| 元组(tuple) | 不可变序列，可哈希 | 不可变数据结构 |
| 集合(set) | 无序不重复，基于哈希 | 集合运算实现 |
| 迭代器 | __iter__, __next__ | 遍历算法基础 |
| 递归 | 函数自调用，栈实现 | 树/图遍历 |
| 装饰器 | @语法，函数包装 | AOP编程思想 |

### 3.3 数据结构与算法对照表

| 数据结构 | 核心操作 | 时间复杂度 | 典型算法/问题 | Python实现 |
|:--------:|:---------|:----------:|:-------------|:-----------|
| 数组 | 访问/插入/删除 | O(1)/O(n)/O(n) | 二分查找、双指针 | list |
| 链表 | 插入/删除/遍历 | O(1)/O(1)/O(n) | 反转、合并、环检测 | 自定义类 |
| 栈 | 入栈/出栈/查看 | O(1)/O(1)/O(1) | 表达式求值、括号匹配 | list/ deque |
| 队列 | 入队/出队 | O(1)/O(1) | BFS、滑动窗口 | deque |
| 哈希表 | 插入/查找/删除 | O(1)/O(1)/O(1) | 去重、计数、缓存 | dict/set |
| 二叉树 | 遍历/查找 | O(n)/O(h) | DFS/BFS遍历、重建 | 自定义类 |
| BST | 查找/插入/删除 | O(h) | 范围查询、第K大 | 自定义类 |
| 堆 | 插入/删除顶 | O(logn)/O(logn) | TopK、堆排序 | heapq |
| 图 | 存储/遍历 | O(V+E) | 最短路径、最小生成树 | dict+list |
