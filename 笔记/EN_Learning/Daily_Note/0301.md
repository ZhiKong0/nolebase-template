# Daily Note 0301

## facing 语法点

```
challenges facing humanity
└─────┬────┘ └────┬────┘
     名词      现在分词（后置定语）
              = 使...面临的/正在面对的
```

**含义**：facing = 使...面临 / 正在面对（主动含义）

**例句**：

- challenges **facing** humanity = 人类**面临的**挑战
- problems **facing** us = 我们**面临的**问题

**注意**：不是"面对人类"，而是"使人类面临的"。

---

## 拓展：现在分词作后置定语

**结构**：名词 + 现在分词（后置定语）

|       现在分词       |   含义   |                 例句                 |             翻译             |
| :------------------: | :-------: | :-----------------------------------: | :--------------------------: |
|   **facing**   | 使...面临 |    the problems**facing** us    |   我们**面临的**问题   |
| **concerning** |   关于   |  issues**concerning** climate  |   **关于**气候的问题   |
| **regarding** |   关于   | questions**re garding** payment |   **关于**付款的问题   |
| **following** | 在...之后 | the meeting**following** lunch |   午餐**之后的**会议   |
| **including** |   包括   |    everyone**including** me    | **包括**我在内的每个人 |
| **involving** |   涉及   |    cases**involving** fraud    |   **涉及**欺诈的案件   |
| **affecting** |   影响   |  factors **affecting** health  |   **影响**健康的因素   |

### 对比：前置定语 vs 后置定语

| 前置定语（不够地道） |         后置定语（更自然）         |
| :------------------: | :---------------------------------: |
| the facing challenge |  the challenge**facing** us  |
| the concerning issue |  the issue**concerning** you  |
|  the following day  | the day **following** Monday |

**规律**：

- 现在分词放**名词后**更自然
- 表达"关于/涉及/影响...的"含义
