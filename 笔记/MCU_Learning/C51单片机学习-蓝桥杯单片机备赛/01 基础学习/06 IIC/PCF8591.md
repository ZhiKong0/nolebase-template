### 二、PCF8591芯片概述

#### 2.1 芯片功能
- **核心功能**：8位AD/DA转换器
- **模拟输入**：4路单端或2路差分输入通道
- **模拟输出**：1路DA转换输出
- **接口类型**：I2C两线串行总线
- **分辨率**：8位（256级，0-255）

#### 2.2 硬件连接（CT107D开发板）

| PCF8591引脚 | 连接 | 功能 |
|-------------|------|------|
| **VCC** | 5V | 电源正 |
| **GND** | GND | 电源地 |
| **SCL** | P20 | I2C时钟线 |
| **SDA** | P21 | I2C数据线 |
| **AIN0** | NC | 通道0（扩展接口/未用） |
| **AIN1** | 光敏电阻 | 通道1（光敏电阻RD1） |
| **AIN2** | NC | 通道2（扩展接口） |
| **AIN3** | 电位器 | 通道3（电位器Rb2） |
| **AOUT** | 测量点 | DA模拟输出 |
| **VREF** | 5V | 参考电压 |

**CT107D连接图**：
```
STC15F2K60S2          PCF8591芯片          外设
    P2.0  ──────────► SCL (时钟)
    P2.1  ◄────————─► SDA (数据)
                      AIN1 ◄──── 光敏电阻RD1
                      AIN3 ◄──── 电位器Rb2
                      AOUT ────► 可测量点
```

### 三、I2C设备地址

| 设备 | 写地址 | 读地址 | 说明 |
|------|--------|--------|------|
| **PCF8591** | 0x90 | 0x91 | 7位地址0x48 + 读写位 |

**地址结构解析**：
```
0x90 = 1001 0000
       │││││││└─ 读写位：0=写
       │││││└┴┴── 器件地址：000
       └┴┴┴────── 固定前缀：1001（PCF8591标识）

0x91 = 1001 0001
                         └─ 读写位：1=读
```

### 四、模拟输入通道

#### 4.1 通道分配
| 通道 | 连接 | 用途 |
|------|------|------|
| **通道0** | 扩展接口 | 未用/外部输入 |
| **通道1** | 光敏电阻RD1 | 测量光照强度 |
| **通道2** | 扩展接口 | 未用/外部输入 |
| **通道3** | 电位器Rb2 | 可调电压输入 |

#### 4.2 光敏电阻原理
- **特性**：光照越强，电阻越小
- **接法**：上拉电阻分压电路
- **输出**：0-255（暗→亮）
  - 暗环境：值较大（接近255）
  - 亮环境：值较小（接近0）

#### 4.3 电位器原理
- **接法**：分压电路，0-5V可调
- **输出**：0-255线性对应
  - 左旋：0（0V）
  - 右旋：255（5V）

### 五、控制字节格式

#### 5.1 控制字各位定义
```
┌────┬────┬────┬────┬────┬────┬────┬────┐
│bit7│bit6│bit5│bit4│bit3│bit2│bit1│bit0│
├────┼────┼────┼────┼────┼────┼────┼────┤
│ 0  │DAEN│ 0  │AIN │CH1 │CH0 │ 0  │ 0  │
└────┴────┴────┴────┴────┴────┴────┴────┘
```

| 位 | 名称 | 功能 |
|----|------|------|
| bit7 | - | 固定为0 |
| bit6 | **DAEN** | 1=使能DA输出，0=禁止DA输出 |
| bit5 | - | 固定为0 |
| bit4 | **AIN** | 1=差分输入，0=单端输入 |
| bit3-bit2 | **CH1-CH0** | 通道选择：00=CH0, 01=CH1, 10=CH2, 11=CH3 |
| bit1-bit0 | - | 固定为00 |

#### 5.2 常用控制字

| 控制字 | 功能 | 说明 |
|--------|------|------|
| `0x00` | 单端AD，通道0 | 读取通道0 |
| `0x01` | 单端AD，通道1 | **读取光敏电阻** |
| `0x02` | 单端AD，通道2 | 读取通道2 |
| `0x03` | 单端AD，通道3 | **读取电位器** |
| `0x40` | 使能DA输出 | 输出DA值 |

### 六、AD转换详解

#### 6.1 转换原理
- **分辨率**：8位（256级）
- **参考电压**：5V
- **分辨率计算**：5V ÷ 256 ≈ 19.5mV/LSB
- **转换公式**：
  ```
  输入电压 = (AD值 / 255) × 5V
  AD值 = (输入电压 / 5V) × 255
  ```

#### 6.2 Dummy Read机制
**重要**：PCF8591需要两次读取，第一次丢弃！

```
原因：
PCF8591内部ADC转换需要时间。
第一次读取的值是上一次转换的结果（可能来自其他通道）。
第二次读取才是当前通道的最新转换结果。

正确流程：
1. 发送控制字选择通道
2. 启动读取
3. 读取第1字节 → 丢弃（Dummy Read）
4. 读取第2字节 → 有效数据
```

### 七、DA转换详解

#### 7.1 转换原理
- **分辨率**：8位（256级）
- **输出电压**：0-5V（与VREF相关）
- **分辨率计算**：5V ÷ 256 ≈ 19.5mV/LSB
- **输出公式**：
  ```
  输出电压 = (DA值 / 255) × 5V
  DA值 = (目标电压 / 5V) × 255
  ```

#### 7.2 输出流程
```
启动 -> 发送地址(0x90) -> ACK -> 发送控制字(0x40使能DA) -> ACK
-> 发送DA值(0-255) -> ACK -> 停止
```

#### 7.3 实测验证
- 可用万用表测量AOUT引脚电压
- 写入0 → 测量约0V
- 写入128 → 测量约2.5V
- 写入255 → 测量约5V

### 八、代码实现

#### 8.1 AD读取函数
```c
/**
 * @brief PCF8591读取指定通道的AD值
 * @param channel 通道号0-3
 *              - 0: 通道0（扩展）
 *              - 1: 通道1（光敏电阻）
 *              - 2: 通道2（扩展）
 *              - 3: 通道3（电位器）
 * @retval 8位AD值0-255
 * @note 采用Dummy Read机制，第一次读取丢弃
 */
u8 PCF8591_Read(u8 channel)
{
    u8 dat;
    
    // 第一次通信：发送控制字选择通道
    I2C_Start();
    I2C_SendByte(0x90);                 // PCF8591写地址
    I2C_WaitAck();
    I2C_SendByte(0x00 | (channel & 0x03));  // 选择通道
    I2C_WaitAck();
    
    // 第二次通信：读取AD值
    I2C_Start();
    I2C_SendByte(0x91);                 // PCF8591读地址
    I2C_WaitAck();
    (void)I2C_RecvByte();               // Dummy Read，丢弃！
    I2C_SendAck(0);                     // ACK，继续读取
    dat = I2C_RecvByte();               // 有效数据
    I2C_SendAck(1);                     // NACK，结束读取
    I2C_Stop();
    
    return dat;
}
```

#### 8.2 DA输出函数
```c
/**
 * @brief PCF8591输出DA值
 * @param dat 8位DA输出值，范围0-255
 * @details 输出模拟电压 = dat / 255 × 5V
 * @note 可用万用表测量AOUT引脚验证
 */
void PCF8591_Write(u8 dat)
{
    I2C_Start();
    I2C_SendByte(0x90);                 // PCF8591写地址
    I2C_WaitAck();
    I2C_SendByte(0x40);                 // 控制字：使能DA输出
    I2C_WaitAck();
    I2C_SendByte(dat);                  // DA输出值
    I2C_WaitAck();
    I2C_Stop();
}
```

### 九、应用实例

#### 9.1 读取光敏电阻并显示
```c
u8 light_value;      // 光敏电阻AD值

void main(void)
{
    System_Init();
    
    while(1)
    {
        // 读取通道1（光敏电阻）
        light_value = PCF8591_Read(1);
        
        // 在数码管显示0-255
        Nixie_DisplayNumber(light_value);
        
        Delay_ms(200);      // 200ms刷新一次
    }
}
```

#### 9.2 电位器控制数码管亮度
```c
u8 adc_value;
u8 brightness_level;

void main(void)
{
    System_Init();
    
    while(1)
    {
        // 读取电位器值
        adc_value = PCF8591_Read(3);
        
        // 将0-255映射到亮度级别0-10
        brightness_level = adc_value / 25;
        if(brightness_level > 10)
            brightness_level = 10;
        
        // 设置数码管亮度
        Nixie_SetBrightness(brightness_level);
        
        // 显示当前AD值
        Nixie_DisplayNumber(adc_value);
    }
}
```

#### 9.3 DA输出可调电压
```c
u8 da_value = 0;

void main(void)
{
    System_Init();
    
    while(1)
    {
        // 输出DA值
        PCF8591_Write(da_value);
        
        // 递增（可用万用表观察电压变化）
        da_value += 10;
        if(da_value > 255)
            da_value = 0;
            
        Delay_ms(500);      // 500ms变化一次
    }
}
```

#### 9.4 AD采集+DA输出（信号处理）
```c
u8 adc_value;
u8 dac_value;

void main(void)
{
    System_Init();
    
    while(1)
    {
        // 读取电位器值
        adc_value = PCF8591_Read(3);
        
        // 数据处理（例如：反相）
        dac_value = 255 - adc_value;
        
        // DA输出
        PCF8591_Write(dac_value);
        
        // 显示
        Nixie_DisplayNumber(adc_value);
```

### 十、关键注意事项

1. **Dummy Read机制**：
   - PCF8591需要两次读取
   - 第一次读取的值必须丢弃
   - 第二次读取才是有效数据

2. **通道切换延时**：
   - 切换通道后需要足够时间完成转换
   - 建议延时或多次Dummy Read

3. **参考电压**：
   - CT107D开发板VREF接5V
   - 满量程对应5V

4. **AD分辨率**：
   - 8位分辨率约19.5mV/LSB
   - 对精度要求高的应用需考虑

5. **DA输出负载**：
   - DA输出驱动能力有限
   - 测量时用高阻抗万用表

6. **光敏电阻特性**：
   - 暗环境值大（接近255）
   - 亮环境值小（接近0）
   - 是反逻辑关系！

7. **电位器方向**：
   - 确认左旋/右旋对应值增减
   - 可能与习惯方向相反

### 十一、常见问题

**Q1: 读取的AD值不稳定？**
- 确认使用了Dummy Read
- 增加采样延时
- 多次采样取平均值
- 检查电源是否稳定

**Q2: DA输出电压不正确？**
- 检查控制字是否为0x40（使能DA）
- 用万用表直接测量AOUT引脚
- 检查VREF电压

**Q3: 光敏电阻值与预期相反？**
- 光敏电阻是反逻辑的：暗=大值，亮=小值
- 需要的话可以在代码中反转

**Q4: 切换通道后读取错误？**
- 切换通道需要足够延时
- 建议连续两次Dummy Read
- 或增加延时函数

**Q5: 如何同时采集多路AD？**
```c
// 轮流采集多个通道
void ReadAllChannels(u8 *values)
{
    for(u8 ch = 0; ch < 4; ch++)
    {
        values[ch] = PCF8591_Read(ch);
        Delay_ms(10);       // 通道间延时
    }
}
```
