# DS18B20 执行时序图与执行过程

## 一、单总线初始化/复位时序

```
主机（单片机）                    从机（DS18B20）

DQ ─────┐                       ┌──────────────────
        │      480-960us      │    应答脉冲
        └─────────────────────┘    60-240us
        ↑                        ↑
     主机拉低                  从机拉低

时序：主机拉低 ≥480us → 释放 → 等待从机应答（拉低60-240us）
```

**关键点**：
- 主机先拉低DQ线480μs以上，产生复位脉冲
- 释放后，DS18B20会拉低60-240μs作为应答（存在脉冲）
- 主机在释放后60-240μs内采样，判断是否存在DS18B20

---

## 二、写时隙（Write Time Slot）

```
写"1"时隙：
DQ ────┐      ┌────────────────────────────
       │ 15us │
       └──────┘
       主机拉低 <15us即释放，表示写"1"

写"0"时隙：
DQ ────┐                                    ┌────
       │              60-120us               │
       └───────────────────────────────────┘
       主机拉低保持60μs以上，表示写"0"
       
最小间隔：1μs
```

**关键点**：
- 写"1"：主机拉低 <15μs即释放
- 写"0"：主机拉低保持60μs以上
- 两个时隙之间最小间隔1μs
- 总时长60-120μs

---

## 三、读时隙（Read Time Slot）

```
DQ ────┐     ┌──────────────────────┐     ┌────
       │1us │  从机输出数据        │
       └───┘                      └─────┘
           ↑
        主机采样（15μs内）

时序：主机拉低1μs → 释放 → 15μs内采样 → 等待至60μs结束
```

**关键点**：
- 主机拉低1μs后释放总线
- 从机在释放后输出数据
- 主机在15μs内采样DQ线电平
- 读"1"：DQ为高；读"0"：DQ为低

## 四、函数一：Delay_OneWire - 单总线延时

### 代码实现

```c
/**
 * @brief 单总线延时函数
 * @param t 延时单位数
 * 
 * 延时原理：嵌套循环实现软件延时
 * - 外层while(t--)循环t次
 * - 内层for循环12次（约12个机器周期）
 * 
 * 延时计算（11.0592MHz晶振）：
 * - 机器周期 = 12 / 11.0592MHz ≈ 1.085μs
 * - for循环12次 ≈ 12 * 1.085μs ≈ 13μs
 * - while循环开销 ≈ 1μs
 * - 总延时 ≈ t * 14μs
 * 
 * 常用延时值：
 * - t=5: 约70μs（写时隙）
 * - t=10: 约140μs（读时隙采样等待）
 * - t=12: 约168μs（初始化前稳定）
 * - t=80: 约1120μs（复位脉冲）
 * - t=200: 约2.8ms（温度转换等待）
 */
void Delay_OneWire(unsigned int t)
{
	unsigned char i;
	while(t--)
	{
		for(i = 0; i < 12; i++);
	}
}
```

### 执行过程

**延时计算（11.0592MHz晶振）**：
```
机器周期 = 12/11.0592MHz ≈ 1.085μs
for循环12次 ≈ 12 * 1.085μs ≈ 13μs
while(t--)执行约1μs

总延时 ≈ t * 14μs

常用延时值：
- t=5: 约70μs（写时隙）
- t=200: 约2.8ms（等待转换）
```

## 五、函数二：init_ds18b20 - 初始化/复位

### 代码实现

```c
bit init_ds18b20(void)
{
    bit initflag = 0;
    
    DQ = 1;              // 1. DQ置高
    Delay_OneWire(12);   // 2. 延时约168μs（等待总线稳定）
    DQ = 0;              // 3. 主机拉低DQ
    Delay_OneWire(80);   // 4. 延时约1120μs（复位脉冲，>480μs）
    DQ = 1;              // 5. 主机释放DQ（上拉电阻拉高）
    Delay_OneWire(10);   // 6. 延时约140μs（等待从机应答）
    initflag = DQ;       // 7. 采样DQ状态
    Delay_OneWire(5);    // 8. 延时约70μs（等待从机释放）
    
    return initflag;     // 返回0=存在，1=不存在
}
```

### 执行过程详解

```
调用：init_ds18b20()

步骤1：初始化
    DQ = 1                     // 确保总线初始为高
    Delay_OneWire(12)          // 延时等待总线稳定

步骤2：发送复位脉冲
    DQ = 0                     // 主机拉低DQ
    Delay_OneWire(80)          // 延时约1120μs（>480μs的复位脉冲）
    
步骤3：释放并等待应答
    DQ = 1                     // 主机释放，上拉电阻拉高
    Delay_OneWire(10)          // 延时约140μs（等待从机拉低应答）
    
步骤4：采样判断
    initflag = DQ              // 读取DQ状态
                               // DQ=0：从机应答（存在）
                               // DQ=1：无应答（不存在）
    
步骤5：等待从机释放
    Delay_OneWire(5)           // 延时约70μs
    
步骤6：返回结果
    return initflag            // 0=成功，1=失败

时序图：
DQ ─────┐    ┌─────────────────────────┐     ┌──────────┐     ┌────
        │168│        1120μs           │ 140 │ 应答60μs │ 70μs│
        └───┘                         └─────┘          └─────┘
        高       低（复位脉冲）        高      低       高
                 主机拉低              上拉   从机拉低  从机释放
```

## 六、函数三：Write_DS18B20 - 写入1字节

### 代码实现

```c
void Write_DS18B20(unsigned char dat)
{
    unsigned char i;
    for(i = 0; i < 8; i++)      // 循环8次，每次写1位
    {
        DQ = 0;                 // 1. 主机拉低DQ（启动写时隙）
        DQ = dat & 0x01;        // 2. 放置数据到DQ（最低位）
        Delay_OneWire(5);     // 3. 延时约70μs（保持时隙）
        DQ = 1;                 // 4. 释放DQ（结束写时隙）
        dat >>= 1;              // 5. 数据右移，准备下一位
    }
    Delay_OneWire(5);           // 6. 额外延时
}
```

### 执行过程示例：写入 0xCC（Skip ROM命令）

```
初始：dat = 0xCC = 1100 1100（二进制）

第1轮（bit0=0）：
    DQ = 0                     // 拉低DQ
    DQ = dat & 0x01 → 0        // 放置bit0=0
    Delay_OneWire(5)           // 延时约70μs（写"0"时隙）
    DQ = 1                     // 释放DQ
    dat >>= 1 → dat = 0110 0110 = 0x66
    
第2轮（bit1=0）：
    DQ = 0
    DQ = 0                     // bit1=0
    Delay_OneWire(5)
    DQ = 1
    dat >>= 1 → dat = 0011 0011 = 0x33
    
第3轮（bit2=1）：
    DQ = 0
    DQ = 1                     // bit2=1
    Delay_OneWire(5)           // 延时约70μs
    DQ = 1                     // 释放（写"1"在<15μs就释放）
    dat >>= 1 → dat = 0001 1001 = 0x19
    
第4轮（bit3=1）：
    DQ = 0
    DQ = 1                     // bit3=1
    Delay_OneWire(5)
    DQ = 1
    dat >>= 1 → dat = 0000 1100 = 0x0C
    
第5轮（bit4=0）：
    DQ = 0
    DQ = 0                     // bit4=0
    Delay_OneWire(5)
    DQ = 1
    dat >>= 1 → dat = 0000 0110 = 0x06
    
第6轮（bit5=0）：
    DQ = 0
    DQ = 0                     // bit5=0
    Delay_OneWire(5)
    DQ = 1
    dat >>= 1 → dat = 0000 0011 = 0x03
    
第7轮（bit6=1）：
    DQ = 0
    DQ = 1                     // bit6=1
    Delay_OneWire(5)
    DQ = 1
    dat >>= 1 → dat = 0000 0001 = 0x01
    
第8轮（bit7=1）：
    DQ = 0
    DQ = 1                     // bit7=1
    Delay_OneWire(5)
    DQ = 1
    dat >>= 1 → dat = 0000 0000 = 0x00

发送顺序：0→0→1→1→0→0→1→1（从低位到高位）
最终数据：0xCC（Skip ROM命令）
```

## 七、函数四：Read_DS18B20 - 读取1字节

### 代码实现

```c
unsigned char Read_DS18B20(void)
{
    unsigned char i;
    unsigned char dat = 0;
    
    for(i = 0; i < 8; i++)      // 循环8次，每次读1位
    {
        DQ = 0;                 // 1. 主机拉低DQ（启动读时隙）
        dat >>= 1;              // 2. 数据右移，腾出最高位
        DQ = 1;                 // 3. 主机释放DQ
        if(DQ)                  // 4. 采样DQ电平
        {
            dat |= 0x80;        // 5. 如果为高，最高位置1
        }
        Delay_OneWire(5);       // 6. 延时完成时隙
    }
    return dat;                 // 返回读取的字节
}
```

### 执行过程示例：假设DS18B20返回 0xAA = 1010 1010

```
初始：dat = 0x00

第1轮（读bit0=0）：
    DQ = 0                     // 拉低DQ
    dat >>= 1 → 0x00           // 右移（还是0）
    DQ = 1                     // 释放DQ
    if(DQ) → 假（DQ=0）         // 采样为低
    dat |= 0x80 → 无操作        // dat保持0x00
    Delay_OneWire(5)           // 延时
    
第2轮（读bit1=1）：
    DQ = 0
    dat >>= 1 → 0x00
    DQ = 1
    if(DQ) → 真（DQ=1）         // 采样为高
    dat |= 0x80 → 0x80          // 最高位置1
    Delay_OneWire(5)
    
第3轮（读bit2=0）：
    DQ = 0
    dat >>= 1 → 0x40            // 0x80 >> 1 = 0x40
    DQ = 1
    if(DQ) → 假
    dat |= 0x80 → 无操作
    Delay_OneWire(5)
    
第4轮（读bit3=1）：
    DQ = 0
    dat >>= 1 → 0x20
    DQ = 1
    if(DQ) → 真
    dat |= 0x80 → 0xA0          // 0x20 | 0x80
    Delay_OneWire(5)
    
第5轮（读bit4=0）：
    DQ = 0
    dat >>= 1 → 0x50
    DQ = 1
    if(DQ) → 假
    Delay_OneWire(5)
    
第6轮（读bit5=1）：
    DQ = 0
    dat >>= 1 → 0x28
    DQ = 1
    if(DQ) → 真
    dat |= 0x80 → 0xA8
    Delay_OneWire(5)
    
第7轮（读bit6=0）：
    DQ = 0
    dat >>= 1 → 0x54
    DQ = 1
    if(DQ) → 假
    Delay_OneWire(5)
    
第8轮（读bit7=1）：
    DQ = 0
    dat >>= 1 → 0x2A
    DQ = 1
    if(DQ) → 真
    dat |= 0x80 → 0xAA          // 最终结果！
    Delay_OneWire(5)

返回值：dat = 0xAA

数据组装过程图解：
初始: dat = 0000 0000

i=0: dat >>= 1 → 0000 0000, DQ=0 → dat = 0000 0000
i=1: dat >>= 1 → 0000 0000, DQ=1 → dat = 1000 0000 (0x80)
i=2: dat >>= 1 → 0100 0000, DQ=0 → dat = 0100 0000 (0x40)
i=3: dat >>= 1 → 0010 0000, DQ=1 → dat = 1010 0000 (0xA0)
i=4: dat >>= 1 → 0101 0000, DQ=0 → dat = 0101 0000 (0x50)
i=5: dat >>= 1 → 0010 1000, DQ=1 → dat = 1010 1000 (0xA8)
i=6: dat >>= 1 → 0101 0100, DQ=0 → dat = 0101 0100 (0x54)
i=7: dat >>= 1 → 0010 1010, DQ=1 → dat = 1010 1010 (0xAA) ✓
```

## 八、完整读取温度流程

### 代码实现

```c
s16 DS18B20_ReadRawTemp(void)
{
    unsigned char low, high;
    s16 temp;
    
    // 阶段1：启动温度转换
    init_ds18b20();              // 初始化
    Write_DS18B20(0xCC);         // Skip ROM
    Write_DS18B20(0x44);         // Convert T（启动转换）
    Delay_OneWire(200);          // 延时等待转换完成（12位约750ms）
    
    // 阶段2：读取温度
    init_ds18b20();              // 再次初始化
    Write_DS18B20(0xCC);         // Skip ROM
    Write_DS18B20(0xBE);         // Read Scratchpad（读取命令）
    
    low = Read_DS18B20();        // 读取低字节
    high = Read_DS18B20();       // 读取高字节
    
    temp = (high << 8) | low;    // 组合成16位有符号数
    return temp;                 // 返回原始温度数据
}
```

### 执行过程详解

```
调用：DS18B20_ReadRawTemp()

【阶段1】启动温度转换
    
    Step 1: init_ds18b20()
    ├── DQ=1, Delay(12)
    ├── DQ=0, Delay(80)         // 复位脉冲
    ├── DQ=1, Delay(10)
    ├── initflag = DQ           // 检测应答
    └── Delay(5)
    
    Step 2: Write_DS18B20(0xCC)  // Skip ROM
    ├── for i=0 to 7:
    │   ├── DQ=0
    │   ├── DQ = bit[i]
    │   ├── Delay(5)
    │   └── DQ=1
    └── 发送完成：0xCC
    
    Step 3: Write_DS18B20(0x44)  // Convert T
    ├── for i=0 to 7:
    │   └── 发送bit[i]
    └── 发送完成：0x44（启动转换）
    
    Step 4: Delay_OneWire(200)   // 延时约2.8ms（实际需要750ms）
    
【阶段2】读取温度
    
    Step 5: init_ds18b20()      // 再次初始化
    
    Step 6: Write_DS18B20(0xCC)  // Skip ROM
    
    Step 7: Write_DS18B20(0xBE)  // Read Scratchpad
    
    Step 8: low = Read_DS18B20()
    ├── for i=0 to 7:
    │   ├── DQ=0
    │   ├── dat >>= 1
    │   ├── DQ=1
    │   ├── if(DQ) dat |= 0x80
    │   └── Delay(5)
    └── 返回低字节（如0x91）
    
    Step 9: high = Read_DS18B20()
    └── 返回高字节（如0x01）
    
    Step 10: 组合数据
    temp = (high << 8) | low
         = (0x01 << 8) | 0x91
         = 0x0191
         = 25.0625°C（二进制补码）

总通信次数：
- 2次初始化（复位）
- 4次写命令（0xCC, 0x44, 0xCC, 0xBE）
- 2次读数据（低字节，高字节）
```

## 九、四个函数对比总结

| 函数 | 功能 | 参数 | 返回值 | 调用场景 |
|:----:|:----:|:----:|:------:|:--------:|
| **Delay_OneWire** | 延时函数 | t（延时单位） | 无 | 时隙延时 |
| **init_ds18b20** | 初始化/复位 | 无 | 0=存在，1=不存在 | 通信前调用 |
| **Write_DS18B20** | 写1字节 | dat（8位数据） | 无 | 发送命令 |
| **Read_DS18B20** | 读1字节 | 无 | 读取的8位数据 | 读取温度 |

**调用关系**：
```
DS18B20_ReadRawTemp()
        │
        ├──── init_ds18b20() ×2
        ├──── Write_DS18B20() ×4
        └──── Read_DS18B20() ×2
```
