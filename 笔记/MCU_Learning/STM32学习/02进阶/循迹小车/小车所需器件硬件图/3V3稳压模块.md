# 3V3稳压模块

![](https://raw.githubusercontent.com/ZhiKong0/Image_Auto/main/Obsidian/6f3eeaf24da50014c933d46fd6fed099.jpg)
## 引脚说明

**该模块主要用于电压转换与稳压，具体的引脚定义如下：**

| **丝印标识** | **功能名称** | **详细说明**              |
| ------------------ | ------------------ | ------------------------------- |
| **VIN+**     | **输入正极** | **电源输入端（正极）**    |
| **GND**      | **输入负极** | **电源输入端（地/负极）** |
| **VOUT+**    | **输出正极** | **稳压输出端（正极）**    |
| **GND**      | **输出负极** | **稳压输出端（地/负极）** |

---

**接线提示：**

* **输入端 (左侧)** **：连接电源输入，注意正负极不要接反。**
* **输出端 (右侧)** **：连接负载，输出稳定的3.3V电压。**
* **GND** **：输入和输出的地线通常是共用的。**
