# 循迹小车接线图（TB6612FNG + 双MG310(GMR编码器) + STM32F103C8T6）

## 1. 总体说明

本接线方案实现：

- 电机A、电机B：TB6612FNG 双路直流电机驱动
- 编码器A、编码器B：两台 MG310（带 GMR 正交编码器）测速
- STM32F103C8T6：
  - TIM1 输出两路 PWM（调速）
  - TIM2 / TIM3 使用硬件编码器模式（正交解码）

关键原则：

- 电池负极、TB6612 GND、STM32 GND、编码器 GND 必须 **共地**
- 编码器电源使用 **3.3V**（与 STM32 逻辑电平一致）

---

## 2. 电源接线（7.4V 电池 + 3V3 稳压模块）

本项目使用 7.4V（2S 锂电）供电时，供电建议分两路：

- **电机电源（高压）**：电池 7.4V → TB6612 的 **VM**
- **逻辑电源（3.3V）**：电池 7.4V → 3V3 稳压模块 → STM32 3.3V 与 TB6612 的 **VCC**

注意：**7.4V 不能直接接 STM32 的 3.3V 引脚**。

### 2.1 3V3 稳压模块接线

稳压模块引脚（以丝印为准）：`VIN+ / GND / VOUT+ / GND`

- 稳压模块 **VIN+** ← 电池正极（7.4V）
- 稳压模块 **GND(输入)** ← 电池负极
- 稳压模块 **VOUT+** → STM32 **3.3V**
- 稳压模块 **GND(输出)** → STM32 **GND**（与输入 GND 通常内部相通）

### 2.2 TB6612 电源脚说明（非常关键）

- TB6612 **VM**：电机供电输入（接电池 7.4V）
- TB6612 **VCC**：逻辑供电输入（接 3.3V 稳压输出）
- TB6612 **GND**：必须与 STM32、稳压模块、电池负极 **共地**

---

## 3. TB6612FNG ↔ STM32 引脚连接（最终版）

### 3.1 供电/地

- TB6612 **VM(1)** → 7.4V/12V 电池正极
- TB6612 **VCC(2)** → STM32 **3.3V**
- TB6612 **GND(3)** → STM32 **GND**
- TB6612 **GND(15)** → STM32 **GND**
- 电池负极 → STM32 **GND**（与 TB6612 / 编码器共地）

### 3.2 使能

- TB6612 **STBY(14)** → STM32 **PB0**
  - `PB0=1`：驱动使能
  - `PB0=0`：驱动待机

### 3.3 电机A通道（A通道方向脚保持不变）

- TB6612 **PWMA(6)** ← STM32 **PA8 (TIM1_CH1)**
- TB6612 **AIN1(7)** ← STM32 **PA4**
- TB6612 **AIN2(8)** ← STM32 **PA5**
- TB6612 **AO1(4)** → 电机A 电机线+
- TB6612 **AO2(5)** → 电机A 电机线-

### 3.4 电机B通道（为释放 TIM3 编码器口，方向脚改为 PB1/PB10）

- TB6612 **PWMB(11)** ← STM32 **PA9 (TIM1_CH2)**
- TB6612 **BIN1(9)** ← STM32 **PB1**
- TB6612 **BIN2(10)** ← STM32 **PB10**
- TB6612 **BO1(12)** → 电机B 电机线+
- TB6612 **BO2(13)** → 电机B 电机线-

---

## 4. MG310（GMR编码器款）电机引脚定义

电机 6Pin 引脚顺序：`[1][2][3][4][5][6]`

| MG310 引脚 | 功能 |
|---:|---|
| 1 | 电机线 + |
| 2 | 编码器地线 GND |
| 3 | 编码器输出 B 相 |
| 4 | 编码器输出 A 相 |
| 5 | 编码器电源 VCC |
| 6 | 电机线 - |

---

## 5. 两台 MG310 的完整接线

### 5.1 电机A（TB6612 A通道 + 编码器A）

#### 电机线（到 TB6612 输出）

- MG310-A **1(电机+)** → TB6612 **AO1(4)**
- MG310-A **6(电机-)** → TB6612 **AO2(5)**

#### 编码器线（到 STM32，TIM2 正交解码）

- MG310-A **5(编码器电源)** → STM32 **3.3V**
- MG310-A **2(编码器地线)** → STM32 **GND**
- MG310-A **4(A相)** → STM32 **PA0 (TIM2_CH1)**
- MG310-A **3(B相)** → STM32 **PA1 (TIM2_CH2)**

### 5.2 电机B（TB6612 B通道 + 编码器B）

#### 电机线（到 TB6612 输出）

- MG310-B **1(电机+)** → TB6612 **BO1(12)**
- MG310-B **6(电机-)** → TB6612 **BO2(13)**

#### 编码器线（到 STM32，TIM3 正交解码）

- MG310-B **5(编码器电源)** → STM32 **3.3V**
- MG310-B **2(编码器地线)** → STM32 **GND**
- MG310-B **4(A相)** → STM32 **PA6 (TIM3_CH1)**
- MG310-B **3(B相)** → STM32 **PA7 (TIM3_CH2)**

## 6. 快速自检建议（上电前/上电后）

- 先只接 **3.3V 逻辑电源**（稳压模块输出到 STM32 3.3V 与 TB6612 VCC）：确认 STM32 供电正常、`PB0(STBY)` 可控
- 再接 **电池到 TB6612 VM**（电机电源）：
  - `PB0=0` 时电机不应动作
  - `PB0=1` 后再给 PWM，电机可控
- 编码器测试：手转轮子/电机轴，计数应随方向正/负变化（取决于你的计数方向设置）

---

## 7. 备注

- 如果发现“正转方向”与你期望相反：
  - 软件交换方向逻辑（推荐），或
  - 对调该电机两根电机线（MG310 的 1/6 对调）
- 若编码器输出异常（抖动/读不到）：优先检查 **共地** 与 **编码器供电是否为 3.3V**
