## 一、主控芯片 MCU

### STM32F103C8T6 (Blue Pill)

**概述**
- STM32F103C8T6 是 STM32F1 系列的入门级 MCU
- 基于 ARM Cortex-M3 内核，72MHz 主频
- 64KB Flash，20KB SRAM
- 支持 SWD/JTAG 调试

**引脚功能**（基于原理图）
- **电源引脚**：VDD (3.3V)、VSS (GND)、VBAT
- **时钟引脚**：OSC_IN/OSC_OUT (8MHz 晶振)
- **调试引脚**：SWDIO (PA13)、SWCLK (PA14)、NRST
- **GPIO**：PA0-PA15、PB0-PB15、PC13-PC15
- **外设引脚**：
  - USART1：PA9 (TX)、PA10 (RX)
  - USART2：PA2 (TX)、PA3 (RX)
  - I2C1：PB6 (SCL)、PB7 (SDA)
  - SPI1：PA5 (SCK)、PA6 (MISO)、PA7 (MOSI)
  - ADC：PA0-PA7
  - PWM：PA8-PA11、PB6-PB9

**使用注意事项**
1. 供电电压：3.3V（不可直接接 5V）
2. USB 接口为 Micro USB，用于供电和调试
3. 板载 LED 连接在 PC13（低电平点亮）
4. 启动模式选择：BOOT0 引脚
   - 低电平：从 Flash 启动（正常运行）
   - 高电平：从系统存储器启动（下载模式）

### STM32F107VC

**概述**
- Connectivity line MCU，基于 ARM Cortex-M3 内核
- 72MHz 主频，1.25 DMIPS/MHz 性能
- Flash：64-256KB，SRAM：64KB

|------|------|

| Flash | 64-256 Kbytes |
| SRAM | 64 Kbytes |
| 工作电压 | 2.0-3.6V |
| GPIO | 最多 80 个 |

**时钟系统**
- 3-25 MHz 外部晶振
- 内部 8MHz RC 振荡器（出厂校准）
- 内部 40kHz RC（支持校准）
- 32kHz RTC 晶振（支持校准）

**模拟外设**
- 2×12-bit ADC，1μs 转换时间，16 通道
- 转换范围：0-3.6V
- 采样率：最高 2 MSPS（交错模式）
- 2×12-bit DAC
- 温度传感器

**定时器**
- 最多 10 个定时器
- 4 个 16 位通用定时器（PWM/输入捕获/编码器）
- 1 个 16 位电机控制 PWM 定时器（死区时间、急停）
- 2 个看门狗定时器（独立/窗口）
- SysTick 定时器（24 位）
- 2 个 16 位基本定时器（驱动 DAC）

**通信接口**

**低功耗模式**
- Sleep、Stop、Standby 模式
- VBAT 供电支持 RTC 和备份寄存器

**调试接口**

- JTAG
- Cortex-M3 ETM（嵌入式跟踪宏单元）

## 二、惯性测量传感器

### ICM-42688-P

**概述**
ICM-42688-P 是 TDK-InvenSense 推出的高精度 6 轴 MEMS 运动追踪器件，集成 3 轴陀螺仪和 3 轴加速度计。

**核心特性**

| 参数 | 规格 |
|------|------|
| 传感器 | 3轴陀螺仪 + 3轴加速度计 |
| 陀螺仪噪声 | 2.8 mdps/√Hz |
| 加速度计噪声 | 70 µg/√Hz |
| 6轴功耗 | 0.88 mA（低噪声模式） |
| 工作电压 | 1.71-3.6V (VDD)，1.71-3.6V (VDDIO) |

**陀螺仪规格**
- 可选量程 (dps)：±15.6 / 31.2 / 62.5 / 125 / 250 / 500 / 1000 / 2000
- 噪声：2.8 mdps/√Hz
- 温度稳定性：±5 mdps/°C
- 分辨率：16/19-bit

**加速度计规格**
- 可选量程 (g)：±2 / 4 / 8 / 16
- 噪声：70 µg/√Hz
- 分辨率：16/18-bit

**通信接口**
- I3C：12.5 MHz
- I2C：1 MHz
- SPI：24 MHz

**先进功能**
- **APEX 运动处理引擎**：
  - 计步器
  - 倾斜检测
  - 敲击检测
  - 运动唤醒
  - 抬手唤醒/睡眠
  - 显著运动检测
- **FIFO**：2KB FIFO，支持 20-bit 数据格式
  - 19-bit 陀螺仪数据
  - 18-bit 加速度计数据
- **可编程数字滤波器**
- **温度传感器**

**应用场景**
- AR/VR 控制器
- 头戴式显示器 (HMD)
- 可穿戴设备
- 运动追踪
- 机器人
- 物联网应用

**封装**
- 2.5×3mm 14-Pin LGA
- 工作温度：-40°C 至 +85°C

## 三、电机驱动芯片

### TB6612FNG

**概述**
TB6612FNG 是东芝推出的双路直流电机驱动 IC，采用低导通电阻的 LD MOS 输出结构。

**关键规格**

| 参数 | 规格 |
|------|------|
| 电源电压 (VM) | 15V (Max) |
| 输出电流 | 1.2A (平均) / 3.2A (峰值) |
| 导通电阻 | 0.5Ω (上下桥 Typ. @ VM ≥ 5V) |
| 封装 | SSOP24 (0.65mm 引脚间距) |

**工作模式**
通过 IN1/IN2 控制信号选择四种模式：

| IN1 | IN2 | 模式 | 说明 |
|-----|-----|------|------|
| H | L | CW | 正转 |
| L | H | CCW | 反转 |
| H | H | Short Brake | 制动 |
| L | L | Stop | 停止 |

**引脚定义**

| 引脚 | 名称 | 类型 | 功能 |
|------|------|------|------|
| 1, 2 | AO1 | O | A 通道输出 1 |
| 3, 4 | PGND1 | - | A 通道功率地 |
| 5, 6 | AO2 | O | A 通道输出 2 |
| 7, 8 | BO2 | O | B 通道输出 2 |
| 9, 10 | PGND2 | - | B 通道功率地 |
| 11, 12 | BO1 | O | B 通道输出 1 |
| 13, 14, 24 | VM1/VM2/VM3 | - | 电机电源 |
| 15 | PWMB | I | B 通道 PWM 输入（内部 200kΩ 下拉） |
| 16 | BIN2 | I | B 通道输入 2（内部 200kΩ 下拉） |
| 17 | BIN1 | I | B 通道输入 1（内部 200kΩ 下拉） |
| 18 | GND | - | 信号地 |
| 19 | STBY | I | "L"=待机模式（内部 200kΩ 下拉） |
| 20 | Vcc | - | 信号电源 |
| 21 | AIN1 | I | A 通道输入 1（内部 200kΩ 下拉） |
| 22 | AIN2 | I | A 通道输入 2（内部 200kΩ 下拉） |
| 23 | PWMA | I | A 通道 PWM 输入（内部 200kΩ 下拉） |

**保护功能**
- 热关断保护 (TSD)
- 低电压检测 (UVLO)

**使用注意事项**
1. MOS 结构对静电敏感，操作时需防静电
2. 使用防静电手环、导电垫和离子风机
3. 保持合理的环境温度和湿度
4. 电机电源 (VM) 与逻辑电源 (Vcc) 分开

**典型应用电路**
```
        VM (7.2V-15V)
          |
    +-----+-----+
    |     |     |
   VM1   VM2   VM3
    |     |     |
+---+---+---+---+---+
| TB6612FNG         |
|                   |
| AIN1 <--- MCU     |
| AIN2 <--- MCU     |
| PWMA <--- MCU PWM |
|                   |
| BIN1 <--- MCU     |
| BIN2 <--- MCU     |
| PWMB <--- MCU PWM |
|                   |
| STBY <--- MCU     |
+--------+----------+
         |
    Motor A / Motor B
```

## 四、电机使用参考

### MG310 电机驱动 (带 PID 反馈控制)

**概述**
MG310 是一款带编码器的直流减速电机，配合 STM32 和 HAL 库可实现精确的 PID 速度/位置控制。

**电机参数**（典型值）
- 额定电压：6-12V
- 减速比：1:30 或 1:50（视型号而定）
- 编码器分辨率：11-13 PPR（脉冲/转）
- 输出轴转速：约 300-500 RPM（视减速比）

**引脚定义**

| 颜色 | 功能 | 连接 |
|------|------|------|
| 红色 | 电机正极 | 驱动器输出 |
| 黑色 | 电机负极 | 驱动器输出 |
| 绿色 | 编码器 A 相 | MCU 定时器输入 |
| 白色 | 编码器 B 相 | MCU 定时器输入 |
| 蓝色 | 编码器 VCC | 3.3V/5V |
| 黄色 | 编码器 GND | GND |

**HAL 库驱动要点**

1. **PWM 输出配置**
   ```c
   // 使用 TIM1/TIM2/TIM3 等输出 PWM
   // 频率：10-20kHz（避免电机啸叫）
   // 模式：PWM Mode 1
   ```

2. **编码器接口配置**
   ```c
   // 使用定时器编码器模式（Encoder Mode）
   // TIM1/TIM2/TIM3/TIM4 支持编码器接口
   // 模式：TI1 + TI2（四倍频）
   ```

3. **PID 控制框架**
   ```c
   typedef struct {
       float Kp, Ki, Kd;
       float error, last_error;
       float integral;
       float output;
   } PID_TypeDef;
   
   float PID_Calculate(PID_TypeDef* pid, float setpoint, float feedback) {
       pid->error = setpoint - feedback;
       pid->integral += pid->error;
       float derivative = pid->error - pid->last_error;
       pid->output = pid->Kp * pid->error 
                   + pid->Ki * pid->integral 
                   + pid->Kd * derivative;
       pid->last_error = pid->error;
       return pid->output;
   }
   ```

4. **控制周期**
   - 建议：10-50ms
   - 使用 TIM 中断或 SysTick

**PID 参数整定建议**

| 现象 | 调整方法 |
|------|----------|
| 响应慢 | 增大 Kp |
| 震荡 | 减小 Kp，增大 Kd |
| 静差大 | 增大 Ki |
| 超调大 | 增大 Kd |

**完整控制流程**
1. 初始化 PWM 和编码器定时器
2. 设置目标速度/位置
3. 读取编码器值计算实际速度
4. PID 计算输出 PWM 占空比
5. 定时重复步骤 3-4

**注意事项**
1. 编码器供电需与 MCU 电平匹配（3.3V 或 5V）
2. 电机电源与逻辑电源分开，避免干扰
3. 添加滤波电容抑制电机噪声
4. 考虑加入速度限幅和积分限幅防止失控
