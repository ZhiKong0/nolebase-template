## 1. 电机概述

### 1.1 电机简介

MG310是一款**带霍尔编码器的直流减速电机**，集成370直流电机和双通道霍尔编码器，广泛应用于智能小车、机器人和自动化设备中。

### 1.2 关键特性

| 参数 | 典型值 | 说明 |
|------|--------|------|
| **额定电压** | 6V / 12V | 常见工作电压 |
| **减速比** | 1:30 / 1:50 / 1:100 | 可选多种减速比 |
| **编码器分辨率** | 13 PPR (脉冲/转) | 电机轴，减速后更高 |
| **输出轴转速** | 300-500 RPM | @ 12V |
| **堵转电流** | 1.5-2.5A | 启动或堵转时 |
| **额定电流** | 0.2-0.5A | 正常工作电流 |
| **输出轴扭矩** | 1-3 kg·cm | 取决于减速比 |

### 1.3 应用场景

- ✅ 智能小车/机器人底盘驱动
- ✅ 摄像头云台转动
- ✅ 自动化设备传送带
- ✅ 机械臂关节驱动
- ✅ 模型车/船动力系统

## 4. 编码器原理与测速

### 4.1 正交编码器原理

```
A相:  ___|‾‾‾‾|___|‾‾‾‾|___|‾‾‾‾|___
        
B相:  ‾‾‾|___|‾‾‾‾|___|‾‾‾‾|___|‾‾‾‾
      
方向: 正转 (A相超前B相90°)

A相:  ‾‾‾|___|‾‾‾‾|___|‾‾‾‾|___|‾‾‾‾
        
B相:  ___|‾‾‾‾|___|‾‾‾‾|___|‾‾‾‾|___
      
方向: 反转 (B相超前A相90°)
```

**四倍频计数**: 每个完整周期A/B相有4个边沿变化，可获4倍精度。

### 4.2 STM32编码器接口模式

STM32定时器支持硬件正交解码：

| 模式 | 计数方式 | 精度 |
|------|----------|------|
| 仅TI1 | A相计数 | 1x |
| 仅TI2 | B相计数 | 1x |
| TI1+TI2 | A/B相都计数 | 2x |
| TI1+TI2 (编码器模式3) | 双边沿计数 | 4x |

### 4.3 测速计算方法

```c
// 编码器参数 (以30:1减速比为例)
#define PPR_MOTOR_SHAFT     13      // 电机轴每转脉冲
#define GEAR_RATIO          30      // 减速比
#define PPR_OUTPUT_SHAFT    (PPR_MOTOR_SHAFT * GEAR_RATIO * 4)  // 390 * 4 = 1560

// 测速计算
typedef struct {
    int32_t encoder_count;      // 编码器计数
    int32_t last_count;         // 上次计数
    float speed_rpm;            // 转速 (RPM)
    float speed_dps;            // 转速 (度/秒)
    uint32_t last_time;         // 上次测量时间
} EncoderData_t;

void CalculateSpeed(EncoderData_t* enc, uint32_t current_time_ms) {
    int32_t delta_count = enc->encoder_count - enc->last_count;
    uint32_t delta_time = current_time_ms - enc->last_time;
    
    if (delta_time > 0) {
        // RPM计算
        // RPM = (delta_count / PPR) / (delta_time / 60000)
        enc->speed_rpm = (float)delta_count * 60000.0f / 
                         (PPR_OUTPUT_SHAFT * delta_time);
        
        // 度/秒计算
        enc->speed_dps = enc->speed_rpm * 6.0f;  // 1 RPM = 6 deg/s
    }
    
    enc->last_count = enc->encoder_count;
    enc->last_time = current_time_ms;
}
```

## 5. STM32 HAL库驱动

### 5.1 定时器编码器模式配置

```c
#include "stm32f1xx_hal.h"

TIM_HandleTypeDef htim2;  // 用于编码器输入
TIM_HandleTypeDef htim1;  // 用于PWM输出

/* 编码器定时器初始化 (TIM2 - PA0/PA1) */
void Encoder_TIM2_Init(void) {
    __HAL_RCC_TIM2_CLK_ENABLE();
    __HAL_RCC_GPIOA_CLK_ENABLE();
    
    // PA0: CH1 (A相), PA1: CH2 (B相)
    GPIO_InitTypeDef GPIO_InitStruct = {0};
    GPIO_InitStruct.Pin = GPIO_PIN_0 | GPIO_PIN_1;
    GPIO_InitStruct.Mode = GPIO_MODE_INPUT;
    GPIO_InitStruct.Pull = GPIO_PULLUP;  // 编码器开漏输出需上拉
    HAL_GPIO_Init(GPIOA, &GPIO_InitStruct);
    
    // TIM2配置为编码器模式
    htim2.Instance = TIM2;
    htim2.Init.Prescaler = 0;  // 不分频
    htim2.Init.CounterMode = TIM_COUNTERMODE_UP;
    htim2.Init.Period = 65535;  // 最大计数
    htim2.Init.ClockDivision = TIM_CLOCKDIVISION_DIV1;
    HAL_TIM_Encoder_Init(&htim2, TIM_ENCODERMODE_TI12);  // 双相计数
    
    // 输入捕获配置
    TIM_Encoder_InitTypeDef sConfig = {0};
    sConfig.EncoderMode = TIM_ENCODERMODE_TI12;
    sConfig.IC1Polarity = TIM_ICPOLARITY_RISING;
    sConfig.IC1Selection = TIM_ICSELECTION_DIRECTTI;
    sConfig.IC1Prescaler = TIM_ICPSC_DIV1;
    sConfig.IC1Filter = 6;  // 数字滤波
    sConfig.IC2Polarity = TIM_ICPOLARITY_RISING;
    sConfig.IC2Selection = TIM_ICSELECTION_DIRECTTI;
    sConfig.IC2Prescaler = TIM_ICPSC_DIV1;
    sConfig.IC2Filter = 6;
    
    HAL_TIM_Encoder_ConfigChannel(&htim2, &sConfig, TIM_CHANNEL_1, TIM_CHANNEL_2);
    
    HAL_TIM_Encoder_Start(&htim2, TIM_CHANNEL_ALL);
}

/* PWM定时器初始化 (TIM1 - PA8) */
void PWM_TIM1_Init(void) {
    __HAL_RCC_TIM1_CLK_ENABLE();
    __HAL_RCC_GPIOA_CLK_ENABLE();
    
    // PA8: TIM1_CH1 (PWM输出)
    GPIO_InitTypeDef GPIO_InitStruct = {0};
    GPIO_InitStruct.Pin = GPIO_PIN_8;
    GPIO_InitStruct.Mode = GPIO_MODE_AF_PP;
    GPIO_InitStruct.Speed = GPIO_SPEED_FREQ_HIGH;
    HAL_GPIO_Init(GPIOA, &GPIO_InitStruct);
    
    // TIM1配置: 10kHz PWM
    htim1.Instance = TIM1;
    htim1.Init.Prescaler = 0;  // 72MHz
    htim1.Init.CounterMode = TIM_COUNTERMODE_UP;
    htim1.Init.Period = 7199;  // 72MHz / 7200 = 10kHz
    htim1.Init.ClockDivision = TIM_CLOCKDIVISION_DIV1;
    htim1.Init.RepetitionCounter = 0;
    HAL_TIM_PWM_Init(&htim1);
    
    // PWM模式
    TIM_OC_InitTypeDef sConfigOC = {0};
    sConfigOC.OCMode = TIM_OCMODE_PWM1;
    sConfigOC.Pulse = 0;  // 初始0%
    sConfigOC.OCPolarity = TIM_OCPOLARITY_HIGH;
    sConfigOC.OCNPolarity = TIM_OCNPOLARITY_HIGH;
    sConfigOC.OCFastMode = TIM_OCFAST_DISABLE;
    HAL_TIM_PWM_ConfigChannel(&htim1, &sConfigOC, TIM_CHANNEL_1);
    
    HAL_TIM_PWM_Start(&htim1, TIM_CHANNEL_1);
}

/* 电机控制引脚初始化 */
void Motor_GPIO_Init(void) {

    GPIO_InitStruct.Pin = GPIO_PIN_4 | GPIO_PIN_5;
    GPIO_InitStruct.Mode = GPIO_MODE_OUTPUT_PP;

    GPIO_InitStruct.Speed = GPIO_SPEED_FREQ_LOW;
```

### 5.2 电机控制函数

```c
/* 设置电机方向和速度
 * speed: -1000 ~ +1000 (对应PWM占空比)
 * 正值: 正转, 负值: 反转
 */
void Motor_SetSpeed(int16_t speed) {
    if (speed > 1000) speed = 1000;
    if (speed < -1000) speed = -1000;
    
    if (speed > 0) {
        // 正转
        HAL_GPIO_WritePin(GPIOA, GPIO_PIN_4, GPIO_PIN_SET);   // AIN1 = 1
        HAL_GPIO_WritePin(GPIOA, GPIO_PIN_5, GPIO_PIN_RESET);  // AIN2 = 0
        __HAL_TIM_SET_COMPARE(&htim1, TIM_CHANNEL_1, speed);
    } else if (speed < 0) {
        // 反转
        HAL_GPIO_WritePin(GPIOA, GPIO_PIN_4, GPIO_PIN_RESET);   // AIN1 = 0
        HAL_GPIO_WritePin(GPIOA, GPIO_PIN_5, GPIO_PIN_SET);    // AIN2 = 1
        __HAL_TIM_SET_COMPARE(&htim1, TIM_CHANNEL_1, -speed);
    } else {
        // 停止 (制动)
        HAL_GPIO_WritePin(GPIOA, GPIO_PIN_4 | GPIO_PIN_5, GPIO_PIN_RESET);
        __HAL_TIM_SET_COMPARE(&htim1, TIM_CHANNEL_1, 0);
    }
}

/* 读取编码器计数值 */
int32_t Encoder_GetCount(void) {
    return (int32_t)__HAL_TIM_GET_COUNTER(&htim2);
}

/* 重置编码器计数 */
void Encoder_Reset(void) {
    __HAL_TIM_SET_COUNTER(&htim2, 0);
}
```

## 6. PID速度闭环控制

### 6.1 PID控制原理

```
         目标速度
            |
            v
    +-------+-------+
    |               |
    v               |
  +---+     +---+   |
  | - |<----+   |   |
  +---+     | Kp|   |
    |       | Ki|--->+---> 电机控制量
    v       | Kd|   |
  实际速度   +---+   |
            |       |
            +-------+
```

### 6.2 PID结构体与初始化

```c
typedef struct {
    float Kp;           // 比例系数
    float Ki;           // 积分系数
    float Kd;           // 微分系数
    
    float setpoint;     // 目标值
    float error;        // 当前误差
    float last_error;   // 上次误差
    float integral;     // 积分累积
    float derivative;   // 微分值
    
    float output;       // PID输出
    float output_max;   // 输出上限
    float output_min;   // 输出下限
    
    uint32_t last_time; // 上次计算时间
} PID_Controller_t;

void PID_Init(PID_Controller_t* pid, float kp, float ki, float kd) {
    pid->Kp = kp;
    pid->Ki = ki;
    pid->Kd = kd;
    pid->setpoint = 0;
    pid->error = 0;
    pid->last_error = 0;
    pid->integral = 0;
    pid->derivative = 0;
    pid->output = 0;
    pid->output_max = 1000;  // 对应最大PWM
    pid->output_min = -1000;
    pid->last_time = HAL_GetTick();
}

void PID_SetSetpoint(PID_Controller_t* pid, float setpoint) {
    pid->setpoint = setpoint;
}
```

### 6.3 PID计算函数

```c
float PID_Update(PID_Controller_t* pid, float feedback) {
    uint32_t now = HAL_GetTick();
    float dt = (now - pid->last_time) / 1000.0f;  // 转换为秒
    if (dt <= 0) dt = 0.001f;  // 防止除零
    
    // 计算误差
    pid->error = pid->setpoint - feedback;
    
    // 比例项
    float P = pid->Kp * pid->error;
    
    // 积分项 (带抗饱和)
    if (pid->output < pid->output_max && pid->output > pid->output_min) {
        pid->integral += pid->error * dt;
    }
    float I = pid->Ki * pid->integral;
    
    // 微分项
    pid->derivative = (pid->error - pid->last_error) / dt;
    float D = pid->Kd * pid->derivative;
    
    // 计算输出
    pid->output = P + I + D;
    
    // 输出限幅
    if (pid->output > pid->output_max) pid->output = pid->output_max;
    if (pid->output < pid->output_min) pid->output = pid->output_min;
    
    // 保存状态
    pid->last_error = pid->error;
    pid->last_time = now;
    
    return pid->output;
}

// 简化版位置式PID (固定周期调用)
float PID_Compute(PID_Controller_t* pid, float input, float dt) {
    float error = pid->setpoint - input;
    pid->integral += error * dt;
    float derivative = (error - pid->last_error) / dt;
    
    float output = pid->Kp * error + 
                   pid->Ki * pid->integral + 
                   pid->Kd * derivative;
    
    // 限幅
    if (output > pid->output_max) output = pid->output_max;
    if (output < pid->output_min) output = pid->output_min;
    
    pid->last_error = error;
    return output;
}
```

### 6.4 速度闭环控制完整实现

```c
typedef struct {
    int32_t encoder_count;
    int32_t last_count;
    float current_speed;      // 当前转速 (RPM)
    float target_speed;       // 目标转速 (RPM)
    int16_t pwm_output;       // PWM输出
    PID_Controller_t pid;     // PID控制器
    uint32_t last_time;
} MotorController_t;

MotorController_t motor;

void MotorController_Init(void) {
    // 初始化硬件
    Encoder_TIM2_Init();
    PWM_TIM1_Init();
    Motor_GPIO_Init();
    
    // 初始化PID
    PID_Init(&motor.pid, 8.0f, 0.5f, 0.1f);  // Kp, Ki, Kd需根据电机调整
    
    motor.encoder_count = 0;
    motor.last_count = 0;
    motor.current_speed = 0;
    motor.target_speed = 0;
    motor.pwm_output = 0;
    motor.last_time = HAL_GetTick();
}

void MotorController_Update(void) {
    uint32_t now = HAL_GetTick();
    uint32_t dt_ms = now - motor.last_time;
    
    // 50Hz控制频率 (20ms)
    if (dt_ms >= 20) {
        // 读取编码器
        motor.encoder_count = Encoder_GetCount();
        
        // 计算速度 (RPM)
        int32_t delta_count = motor.encoder_count - motor.last_count;
        float dt_s = dt_ms / 1000.0f;
        // 减速后分辨率: 1560 counts/rev (30:1减速比)
        motor.current_speed = (delta_count / 1560.0f) * 60.0f / dt_s;
        
        // PID计算
        motor.pwm_output = (int16_t)PID_Compute(&motor.pid, 
                                                 motor.current_speed, 
                                                 dt_s);
        
        // 输出到电机
        Motor_SetSpeed(motor.pwm_output);
        
        // 保存状态
        motor.last_count = motor.encoder_count;
        motor.last_time = now;
    }
}

void MotorController_SetSpeed(float rpm) {
    motor.target_speed = rpm;
    PID_SetSetpoint(&motor.pid, rpm);
}
```

### 6.5 增量式PID (更稳定)

```c
typedef struct {
    float Kp, Ki, Kd;
    float error, last_error, prev_error;
    float output;
    float output_max, output_min;
} IncrementalPID_t;

float IncrementalPID_Update(IncrementalPID_t* pid, float setpoint, float feedback) {
    pid->error = setpoint - feedback;
    
    float delta_output = pid->Kp * (pid->error - pid->last_error) +
                         pid->Ki * pid->error +
                         pid->Kd * (pid->error - 2*pid->last_error + pid->prev_error);
    
    pid->output += delta_output;
    
    // 限幅
    if (pid->output > pid->output_max) pid->output = pid->output_max;
    if (pid->output < pid->output_min) pid->output = pid->output_min;
    
    pid->prev_error = pid->last_error;
    pid->last_error = pid->error;
    
    return pid->output;
}
```

```c
typedef struct {
    float target_position;    // 目标位置 (转数)
    float current_position;   // 当前位置
    PID_Controller_t position_pid;
    PID_Controller_t speed_pid;

void PositionController_Update(PositionController_t* ctrl) {
    // 读取当前位置
    int32_t count = Encoder_GetCount();

    Motor_SetSpeed((int16_t)pwm);
```

```c
// 差速控制
void DifferentialDrive_SetSpeed(DifferentialDrive_t* drive, 
                                 float linear, float angular) {
    // linear: 线速度 (m/s)
    // angular: 角速度 (rad/s)
    
    float wheel_base = 0.15f;  // 轮距 15cm
    
    float left_speed = linear - angular * wheel_base / 2.0f;

    float left_rpm = left_speed * 60.0f / (PI * wheel_diameter);
    float right_rpm = right_speed * 60.0f / (PI * wheel_diameter);
    
    MotorController_SetSpeed(&drive->left_motor, left_rpm);
    MotorController_SetSpeed(&drive->right_motor, right_rpm);
}
```

```c
    SystemClock_Config();
    
    // 初始化
    MotorController_Init();
    
    // 串口初始化用于调试
    USART1_Init(115200);
    
    printf("MG310 Motor Control Started\r\n");
    
    // 设置目标速度: 100 RPM
    MotorController_SetSpeed(100.0f);
    
    while (1) {
        // 20ms周期控制
        MotorController_Update();
        
        // 每100ms打印一次状态
        static uint32_t last_print = 0;
        if (HAL_GetTick() - last_print >= 100) {
            printf("Target: %.1f RPM, Current: %.1f RPM, PWM: %d\r\n",
                   motor.target_speed, motor.current_speed, motor.pwm_output);
            last_print = HAL_GetTick();
        }
        
        HAL_Delay(5);  // 简短延时
    }
}
```

## 8. 常见问题与故障排除

### 8.1 常见问题

| 问题 | 可能原因 | 解决方法 |
|------|----------|----------|
| 电机不转 | 供电不足 | 检查电池电压，确保>6V |
| | 接线错误 | 检查M+/M-接线 |
| | PWM未输出 | 检查TIM配置 |
| 速度测不准 | 编码器接线松 | 检查A/B相连接 |
| | 无外部上拉 | 编码器开漏输出需上拉 |
| PID震荡 | Kp过大 | 减小Kp，增加Kd |
| | 采样周期不稳定 | 使用定时中断 |
| 电机发热 | 频繁换向 | 增加死区时间 |
| | 堵转 | 检查负载是否过重 |
| 编码器计数乱跳 | 信号干扰 | 缩短走线，加滤波电容 |
| | 接触不良 | 检查接线端子 |

### 8.2 PID调试技巧

1. **先调P**: 从小到大增加Kp直到出现震荡，然后回退50%
2. **再加I**: 从小开始增加Ki消除稳态误差
3. **最后D**: 增加Kd抑制超调，但不要过大导致噪声放大

**典型参数范围**:
- Kp: 5-20
- Ki: 0.1-2.0
- Kd: 0.01-0.5

### 8.3 保护措施

```c
// 过流保护
void Motor_SafetyCheck(void) {
    static int16_t last_pwm = 0;
    
    // 堵转检测 (速度为0但PWM很大)
    if (motor.pwm_output > 800 && fabs(motor.current_speed) < 5.0f) {
        // 可能堵转，降低PWM
        Motor_SetSpeed(0);
        printf("Stall detected! Motor stopped.\r\n");
    }
    
    // 换向死区
    if ((last_pwm > 0 && motor.pwm_output < 0) ||
        (last_pwm < 0 && motor.pwm_output > 0)) {
        Motor_SetSpeed(0);  // 先停止
        HAL_Delay(50);      // 死区时间
    }
    
    last_pwm = motor.pwm_output;
}
```
