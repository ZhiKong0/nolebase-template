## 1. 芯片概述

### 1.1 芯片简介

MPU-6000和MPU-6050是InvenSense推出的**6轴运动追踪传感器**，集成3轴陀螺仪和3轴加速度计，是全球首款集成DMP（数字运动处理器）的6轴传感器。

### 1.2 芯片差异

| 特性 | MPU-6000 | MPU-6050 |
|------|----------|----------|
| **接口** | I2C + SPI | I2C only |
| **I2C地址** | 0x68 / 0x69 | 0x68 / 0x69 |
| **最大速率** | I2C: 400kHz, SPI: 20MHz | I2C: 400kHz |
| **封装** | QFN-24 (4×4×0.9mm) | QFN-24 (4×4×0.9mm) |
| **应用** | 需要SPI高速传输 | 标准I2C应用 |

### 1.3 关键特性

| 参数 | 规格 |
|------|------|
| **陀螺仪量程** | ±250, ±500, ±1000, ±2000 °/sec |
| **加速度计量程** | ±2g, ±4g, ±8g, ±16g |
| **陀螺仪灵敏度** | 131 LSB/°/s (±250dps) |
| **加速度计灵敏度** | 16384 LSB/g (±2g) |
| **数字输出** | 16-bit signed |
| **FIFO** | 1024字节 |
| **温度传感器** | 内置 |
| **工作电压** | VDD: 2.375-3.46V, VDDIO: 1.71-3.46V |
| **功耗** | 3.8mA (陀螺+加速度) / 5μA (睡眠) |

### 4.2 关键寄存器详解

#### 4.2.1 PWR_MGMT_1 (0x6B) - 电源管理1

| Bit | 名称 | 功能 |
|-----|------|------|
| 7 | DEVICE_RESET | 1=复位所有寄存器到默认值 |
| 6 | SLEEP | 1=进入睡眠模式 |
| 5 | CYCLE | 1=周期唤醒模式 |
| 4 | - | 保留 |
| 3 | TEMP_DIS | 1=禁用温度传感器 |
| 2:0 | CLKSEL[2:0] | 时钟源选择 |

**时钟源选择**:
```
000: 内部20MHz振荡器
001: 选择PLL (推荐)
010-100: 保留
101: 停止时钟 (保留)
110-111: 保留
```

#### 4.2.2 GYRO_CONFIG (0x1B) - 陀螺仪配置

| Bit | 名称 | 功能 |
|-----|------|------|
| 7:5 | XG_ST/YG_ST/ZG_ST | 各轴自检使能 |
| 4:3 | FS_SEL[1:0] | 满量程选择 |
| 2:0 | - | 保留 |

**满量程选择**:
```
00: ±250 °/s  (灵敏度: 131 LSB/°/s)
01: ±500 °/s  (灵敏度: 65.5 LSB/°/s)
10: ±1000 °/s (灵敏度: 32.8 LSB/°/s)
11: ±2000 °/s (灵敏度: 16.4 LSB/°/s)
```

#### 4.2.3 ACCEL_CONFIG (0x1C) - 加速度计配置

| Bit | 名称 | 功能 |
|-----|------|------|
| 7:5 | XA_ST/YA_ST/ZA_ST | 各轴自检使能 |
| 4:3 | AFS_SEL[1:0] | 满量程选择 |
| 2:0 | - | 保留 |

**满量程选择**:
```
00: ±2g   (灵敏度: 16384 LSB/g)
01: ±4g   (灵敏度: 8192 LSB/g)
10: ±8g   (灵敏度: 4096 LSB/g)
11: ±16g  (灵敏度: 2048 LSB/g)
```

#### 4.2.4 CONFIG (0x1A) - 配置

| Bit | 名称 | 功能 |
|-----|------|------|
| 7 | - | 保留 |
| 6 | EXT_SYNC_SET[2] | 外部帧同步 |
| 5 | EXT_SYNC_SET[1] | |
| 4 | EXT_SYNC_SET[0] | |
| 3:0 | DLPF_CFG[3:0] | 数字低通滤波配置 |

**DLPF_CFG** (陀螺仪低通滤波):

## 5. 初始化与配置

### 5.1 标准初始化流程

```c
#define MPU6050_ADDR 0xD0  // I2C地址 (0x68 << 1)

// 寄存器地址
#define MPU6050_REG_PWR_MGMT_1    0x6B
#define MPU6050_REG_SMPLRT_DIV    0x19
#define MPU6050_REG_CONFIG        0x1A
#define MPU6050_REG_GYRO_CONFIG   0x1B
#define MPU6050_REG_ACCEL_CONFIG  0x1C
#define MPU6050_REG_FIFO_EN       0x23
#define MPU6050_REG_INT_ENABLE    0x38
#define MPU6050_REG_WHO_AM_I      0x75

void MPU6050_Init(void) {
    // 1. 检查设备ID
    uint8_t whoami = MPU6050_ReadReg(MPU6050_REG_WHO_AM_I);
    if (whoami != 0x68 && whoami != 0x72) {
        // 设备ID错误
        return ERROR;
    }
    
    // 2. 复位设备
    MPU6050_WriteReg(MPU6050_REG_PWR_MGMT_1, 0x80);  // DEVICE_RESET
    HAL_Delay(100);
    
    // 3. 唤醒设备，选择PLL时钟源
    MPU6050_WriteReg(MPU6050_REG_PWR_MGMT_1, 0x01);  // CLKSEL=1 (PLL)
    HAL_Delay(10);
    
    // 4. 配置采样率
    // 采样率 = 陀螺仪输出速率 / (1 + SMPLRT_DIV)
    // 1kHz / (1+0) = 1kHz
    MPU6050_WriteReg(MPU6050_REG_SMPLRT_DIV, 0x00);
    
    // 5. 配置低通滤波
    MPU6050_WriteReg(MPU6050_REG_CONFIG, 0x03);  // DLPF_CFG=3, 44Hz带宽
    
    // 6. 配置陀螺仪量程 (±2000°/s)
    MPU6050_WriteReg(MPU6050_REG_GYRO_CONFIG, 0x18);  // FS_SEL=3
    
    // 7. 配置加速度计量程 (±2g)
    MPU6050_WriteReg(MPU6050_REG_ACCEL_CONFIG, 0x00);  // AFS_SEL=0
    
    // 8. 使能数据就绪中断 (可选)
    MPU6050_WriteReg(MPU6050_REG_INT_ENABLE, 0x01);
}
```

### 5.2 低功耗配置

```c
void MPU6050_LowPower_Init(void) {
    // 复位
    MPU6050_WriteReg(MPU6050_REG_PWR_MGMT_1, 0x80);
    HAL_Delay(100);
    
    // 仅使能加速度计，进入低功耗模式
    MPU6050_WriteReg(MPU6050_REG_PWR_MGMT_1, 0x20);  // CYCLE=1, SLEEP=0
    
    // 配置唤醒频率 (LP_WAKE_CTRL)
    // 0: 1.25Hz, 1: 2.5Hz, 2: 5Hz, 3: 10Hz
    MPU6050_WriteReg(MPU6050_REG_PWR_MGMT_2, 0x07);  // LP_WAKE_CTRL=0
}
```

## 6. 数据读取与处理

### 6.1 原始数据读取

```c
typedef struct {
    int16_t accel_x;
    int16_t accel_y;
    int16_t accel_z;
    int16_t gyro_x;
    int16_t gyro_y;
    int16_t gyro_z;
    int16_t temperature;
} MPU6050_Data_t;

void MPU6050_ReadData(MPU6050_Data_t* data) {
    uint8_t buffer[14];
    
    // 批量读取加速度、温度、陀螺仪 (14字节)
    MPU6050_ReadBurst(0x3B, buffer, 14);
    
    // 解析数据 (大端格式)
    data->accel_x = (buffer[0] << 8) | buffer[1];
    data->accel_y = (buffer[2] << 8) | buffer[3];
    data->accel_z = (buffer[4] << 8) | buffer[5];
    data->temperature = (buffer[6] << 8) | buffer[7];
    data->gyro_x = (buffer[8] << 8) | buffer[9];
    data->gyro_y = (buffer[10] << 8) | buffer[11];
    data->gyro_z = (buffer[12] << 8) | buffer[13];
}

// 温度转换
temperature = (raw_temp / 340.0) + 36.53;  // °C
```

### 6.2 转换为物理单位

```c
// 根据配置的量程换算
void MPU6050_ConvertData(MPU6050_Data_t* raw, 
                          float* accel_g, 
                          float* gyro_dps) {
    // 加速度计: 假设±2g配置
    const float accel_scale = 1.0f / 16384.0f;
    accel_g[0] = raw->accel_x * accel_scale;
    accel_g[1] = raw->accel_y * accel_scale;
    accel_g[2] = raw->accel_z * accel_scale;
    
    // 陀螺仪: 假设±2000dps配置
    const float gyro_scale = 1.0f / 16.4f;
    gyro_dps[0] = raw->gyro_x * gyro_scale;
    gyro_dps[1] = raw->gyro_y * gyro_scale;
    gyro_dps[2] = raw->gyro_z * gyro_scale;
}
```

## 7. DMP数字运动处理器

### 7.1 DMP特性

- **片上处理**: 解放主控MCU
- **9轴融合**: 支持外接磁力计
- **输出格式**: 四元数、欧拉角、旋转矩阵
- **自动校准**: 陀螺仪零偏自动校准
- **更新率**: 最高200Hz

### 7.2 DMP固件加载

DMP需要加载固件才能工作，固件数据较大(约3KB)，通常存储在MCU Flash中。

```c
// DMP固件加载 (简化示例)
void MPU6050_DMP_Init(void) {
    // 1. 标准初始化
    MPU6050_Init();
    
    // 2. 写入DMP固件 (dmp_memory数组包含固件数据)
    for (uint16_t i = 0; i < DMP_CODE_SIZE; i += 16) {
        uint8_t bank = i / 256;
        uint8_t address = i % 256;
        MPU6050_WriteMem(bank, address, 16, &dmp_memory[i]);
    }
    
    // 3. 配置DMP
    MPU6050_WriteReg(0x6A, 0x04);  // USER_CTRL: DMP_EN=1, FIFO_EN=1
    
    // 4. 使能DMP中断
    MPU6050_WriteReg(0x38, 0x02);  // INT_ENABLE: DMP_INT_EN=1
}

// 读取DMP四元数输出
void MPU6050_DMP_ReadQuaternion(float* q) {
    uint8_t fifo_buffer[42];
    uint16_t fifo_count;
    
    // 读取FIFO计数
    fifo_count = MPU6050_ReadReg(0x72) << 8;
    fifo_count |= MPU6050_ReadReg(0x73);
    
    if (fifo_count >= 42) {
        // 读取FIFO数据
        MPU6050_ReadBurst(0x74, fifo_buffer, 42);
        
        // 解析四元数 (Q30格式)
        int32_t q0 = (fifo_buffer[0] << 24) | (fifo_buffer[1] << 16) | 
                     (fifo_buffer[2] << 8) | fifo_buffer[3];
        int32_t q1 = (fifo_buffer[4] << 24) | (fifo_buffer[5] << 16) | 
                     (fifo_buffer[6] << 8) | fifo_buffer[7];
        // ... q2, q3
        
        // 转换为浮点数 (除以2^30)
        q[0] = q0 / 1073741824.0f;
        q[1] = q1 / 1073741824.0f;
        // ...
    }
}
```

## 8. 姿态解算

### 8.1 互补滤波 (无DMP时)

```c
typedef struct {
    float roll;
    float pitch;
    float yaw;
} Attitude_t;

// 互补滤波姿态解算
void ComplementaryFilter(MPU6050_Data_t* data, Attitude_t* att, float dt) {
    float accel_x = data->accel_x / 16384.0f;
    float accel_y = data->accel_y / 16384.0f;
    float accel_z = data->accel_z / 16384.0f;
    
    float gyro_x = data->gyro_x / 131.0f;  // °/s
    float gyro_y = data->gyro_y / 131.0f;
    
    // 从加速度计算角度
    float acc_roll = atan2(accel_y, accel_z) * 180.0f / M_PI;
    float acc_pitch = atan2(-accel_x, sqrt(accel_y*accel_y + accel_z*accel_z)) 
                      * 180.0f / M_PI;
    
    // 互补滤波 (alpha = 0.98)
    const float alpha = 0.98f;
    att->roll = alpha * (att->roll + gyro_x * dt) + (1-alpha) * acc_roll;
    att->pitch = alpha * (att->pitch + gyro_y * dt) + (1-alpha) * acc_pitch;
}
```

### 8.2 四元数转欧拉角

```c
void QuaternionToEuler(float* q, Attitude_t* att) {
    float q0 = q[0], q1 = q[1], q2 = q[2], q3 = q[3];
    
    // Roll (x-axis rotation)
    float sinr = 2.0f * (q0 * q1 + q2 * q3);
    float cosr = 1.0f - 2.0f * (q1 * q1 + q2 * q2);
    att->roll = atan2(sinr, cosr) * 180.0f / M_PI;
    
    // Pitch (y-axis rotation)
    float sinp = 2.0f * (q0 * q2 - q3 * q1);
    if (fabs(sinp) >= 1.0f)
        att->pitch = copysign(90.0f, sinp);
    else
        att->pitch = asin(sinp) * 180.0f / M_PI;
    
    // Yaw (z-axis rotation)
    float siny = 2.0f * (q0 * q3 + q1 * q2);
    float cosy = 1.0f - 2.0f * (q2 * q2 + q3 * q3);
    att->yaw = atan2(siny, cosy) * 180.0f / M_PI;
}
```

## 9. 代码示例

### 9.1 完整驱动代码 (HAL库)

```c
#include "stm32f1xx_hal.h"

#define MPU6050_ADDR 0xD0  // I2C写地址 (0x68 << 1)

I2C_HandleTypeDef hi2c1;

// 底层I2C读写
static void MPU6050_WriteReg(uint8_t reg, uint8_t data) {
    uint8_t buf[2] = {reg, data};
    HAL_I2C_Master_Transmit(&hi2c1, MPU6050_ADDR, buf, 2, 100);
}

static uint8_t MPU6050_ReadReg(uint8_t reg) {
    uint8_t data;
    HAL_I2C_Master_Transmit(&hi2c1, MPU6050_ADDR, &reg, 1, 100);
    HAL_I2C_Master_Receive(&hi2c1, MPU6050_ADDR, &data, 1, 100);
    return data;
}

static void MPU6050_ReadBurst(uint8_t reg, uint8_t* data, uint16_t len) {
    HAL_I2C_Master_Transmit(&hi2c1, MPU6050_ADDR, &reg, 1, 100);
    HAL_I2C_Master_Receive(&hi2c1, MPU6050_ADDR, data, len, 100);
}

// 初始化
uint8_t MPU6050_Init(void) {
    // 检查WHO_AM_I
    uint8_t id = MPU6050_ReadReg(0x75);
    if (id != 0x68 && id != 0x72) {
        return 1;  // 错误
    }
    
    // 复位
    MPU6050_WriteReg(0x6B, 0x80);
    HAL_Delay(100);
    
    // 配置时钟源
    MPU6050_WriteReg(0x6B, 0x01);
    HAL_Delay(10);
    
    // 配置采样率
    MPU6050_WriteReg(0x19, 0x07);  // 125Hz (1kHz/(1+7))
    
    // 配置低通滤波
    MPU6050_WriteReg(0x1A, 0x06);  // 5Hz带宽
    
    // 配置陀螺仪 (±250°/s)
    MPU6050_WriteReg(0x1B, 0x00);
    
    // 配置加速度计 (±2g)
    MPU6050_WriteReg(0x1C, 0x00);
    
    // 使能数据就绪中断
    MPU6050_WriteReg(0x38, 0x01);
    
    return 0;
}

// 读取数据
void MPU6050_ReadAll(int16_t* accel, int16_t* gyro, float* temp) {
    uint8_t buf[14];
    MPU6050_ReadBurst(0x3B, buf, 14);
    
    accel[0] = (buf[0] << 8) | buf[1];
    accel[1] = (buf[2] << 8) | buf[3];
    accel[2] = (buf[4] << 8) | buf[5];
    
    int16_t t = (buf[6] << 8) | buf[7];
    *temp = (t / 340.0f) + 36.53f;
    
    gyro[0] = (buf[8] << 8) | buf[9];
    gyro[1] = (buf[10] << 8) | buf[11];
    gyro[2] = (buf[12] << 8) | buf[13];
}

// 主程序
int main(void) {
    HAL_Init();
    SystemClock_Config();
    
    // I2C初始化
    hi2c1.Instance = I2C1;
    hi2c1.Init.ClockSpeed = 400000;
    hi2c1.Init.DutyCycle = I2C_DUTYCYCLE_2;
    hi2c1.Init.OwnAddress1 = 0;
    hi2c1.Init.AddressingMode = I2C_ADDRESSINGMODE_7BIT;
    HAL_I2C_Init(&hi2c1);
    
    // MPU6050初始化
    if (MPU6050_Init() != 0) {
        printf("MPU6050 init failed!\r\n");
    }
    
    int16_t accel[3], gyro[3];
    float temp;
    
    while (1) {
        MPU6050_ReadAll(accel, gyro, &temp);
        
        printf("Accel: %d, %d, %d | Gyro: %d, %d, %d | Temp: %.1f\r\n",
               accel[0], accel[1], accel[2],
               gyro[0], gyro[1], gyro[2], temp);
        
        HAL_Delay(100);
    }
}
```

### B. 常见问题

| 问题 | 原因 | 解决 |
|------|------|------|
| I2C无响应 | 地址错误 | 检查AD0引脚状态 |
| | 上拉缺失 | I2C需4.7kΩ上拉 |
| 数据全为0 | 未退出睡眠 | 写PWR_MGMT_1=0x01 |
| 温度异常 | 未等待稳定 | 复位后等待100ms |
| 漂移严重 | 未校准 | 进行零偏校准 |
| DMP不工作 | 固件未加载 | 检查固件加载过程 |
