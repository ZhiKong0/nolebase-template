## 1. 芯片概述

### 1.1 关键特性

| 参数 | 规格 |
|------|------|
| **传感器类型** | 3轴陀螺仪 + 3轴加速度计 |
| **陀螺仪噪声** | 2.8 mdps/√Hz |
| **加速度计噪声** | 70 µg/√Hz (Z轴), 65 µg/√Hz (XY轴) |
| **6轴功耗** | 0.88 mA (低噪声模式) |
| **接口** | I3C (12.5MHz) / I2C (1MHz) / SPI (24MHz) |
| **FIFO** | 2KB，支持20-bit高精度数据格式 |
| **封装** | 2.5×3×0.91 mm LGA-14 |
| **工作温度** | -40°C 至 +85°C |

### 1.2 陀螺仪规格

| 参数 | 值 | 说明 |
|------|-----|------|
| **量程** | ±15.625 ~ ±2000 dps | 8档可选 |
| **分辨率** | 16/19-bit | 取决于模式和FIFO配置 |
| **噪声** | 2.8 mdps/√Hz | 业界领先 |
| **温度稳定性** | ±5 mdps/°C | 高精度应用 |
| **ODR范围** | 12.5Hz ~ 32kHz | 16档可选 |

**量程设置 (GYRO_FS_SEL)**:

| **噪声** | 65-70 µg/√Hz | XY轴65, Z轴70 |
| **ODR范围** | 1.5625Hz ~ 32kHz | LP/LN模式不同 |

**量程设置 (ACCEL_FS_SEL)**:

## 4. 寄存器映射详解

### 4.1 寄存器Bank系统

ICM-42688-P使用寄存器Bank机制来扩展寄存器空间：

| Bank | 用途 | 访问方式 |
|------|------|----------|
| Bank 0 | 主要配置和数据寄存器 | 默认，REG_BANK_SEL=0 |
| Bank 1 | 陀螺仪高级配置 | REG_BANK_SEL=1 |
| Bank 2 | 加速度计高级配置 | REG_BANK_SEL=2 |
| Bank 4 | APEX配置和校准 | REG_BANK_SEL=4 |

**切换Bank**:
```c
// 写入Bank 0的0x76寄存器 (REG_BANK_SEL)
// Bit 1:0 = BANK_SEL
// 00=Bank0, 01=Bank1, 10=Bank2, 11=Bank4
```

### 4.2 Bank 0 - 核心寄存器

#### 4.2.1 设备配置 (0x11 - DEVICE_CONFIG)

| Bit | 名称 | 功能 |
|-----|------|------|
| 4 | SPI_MODE | 0: Mode 0/3 (默认), 1: Mode 1/2 |
| 0 | SOFT_RESET_CONFIG | 1: 软件复位，等待1ms |

**软件复位流程**:
```c
write_register(0x11, 0x01);  // 触发复位
delay_ms(1);                  // 等待1ms
// 设备恢复默认状态
```

#### 4.2.2 电源管理 (0x4E - PWR_MGMT0)

| Bit | 名称 | 功能 |
|-----|------|------|
| 5 | TEMP_DIS | 0: 温度传感器使能, 1: 禁用 |
| 4 | IDLE | 0: RC振荡器随传感器关闭, 1: 保持运行 |
| 3:2 | GYRO_MODE | 00:关闭, 01:待机, 11:低噪声模式 |
| 1:0 | ACCEL_MODE | 00:关闭, 10:低功耗, 11:低噪声 |

**模式切换注意事项**:
- 陀螺仪从OFF到ON后，需等待45ms才能稳定
- 模式切换后，200µs内不要写入其他寄存器

#### 4.2.3 陀螺仪配置 (0x4F - GYRO_CONFIG0)

| Bit | 名称 | 功能 |
|-----|------|------|
| 7:5 | GYRO_FS_SEL | 量程选择 (见1.2节) |
| 3:0 | GYRO_ODR | 输出数据率 |

**ODR设置**:
```
0000: Reserved
0001: 32kHz
0010: 16kHz
0011: 8kHz
0100: 4kHz
0101: 2kHz
0110: 1kHz (默认)
0111: 200Hz
1000: 100Hz
1001: 50Hz
1010: 25Hz
1011: 12.5Hz
11xx: Reserved
1111: 500Hz
```

#### 4.2.4 加速度计配置 (0x50 - ACCEL_CONFIG0)

| Bit | 名称 | 功能 |
|-----|------|------|
| 7:5 | ACCEL_FS_SEL | 量程选择 (见1.3节) |
| 3:0 | ACCEL_ODR | 输出数据率 |

**注意**: LP模式不支持ODR>500Hz, LN模式不支持ODR<12.5Hz

#### 4.2.5 传感器数据寄存器 (只读)

| 寄存器 | 地址 | 内容 |
|--------|------|------|
| ACCEL_DATA_X1 | 0x1F | X轴加速度高字节 [15:8] |
| ACCEL_DATA_X0 | 0x20 | X轴加速度低字节 [7:0] |
| ACCEL_DATA_Y1 | 0x21 | Y轴加速度高字节 [15:8] |
| ACCEL_DATA_Y0 | 0x22 | Y轴加速度低字节 [7:0] |
| ACCEL_DATA_Z1 | 0x23 | Z轴加速度高字节 [15:8] |
| ACCEL_DATA_Z0 | 0x24 | Z轴加速度低字节 [7:0] |
| GYRO_DATA_X1 | 0x25 | X轴陀螺仪高字节 [15:8] |
| GYRO_DATA_X0 | 0x26 | X轴陀螺仪低字节 [7:0] |
| GYRO_DATA_Y1 | 0x27 | Y轴陀螺仪高字节 [15:8] |
| GYRO_DATA_Y0 | 0x28 | Y轴陀螺仪低字节 [7:0] |
| GYRO_DATA_Z1 | 0x29 | Z轴陀螺仪高字节 [15:8] |
| GYRO_DATA_Z0 | 0x2A | Z轴陀螺仪低字节 [7:0] |
| TEMP_DATA1 | 0x1D | 温度高字节 [15:8] |
| TEMP_DATA0 | 0x1E | 温度低字节 [7:0] |

**温度转换公式**:
```c
// 从传感器寄存器读取
temperature = (temp_data / 132.48) + 25;  // °C

// 从FIFO读取 (8-bit)
temperature = (fifo_temp_data / 2.07) + 25;  // °C
```

#### 4.2.6 中断状态 (0x2D - INT_STATUS)

| Bit | 名称 | 说明 |
|-----|------|------|
| 6 | UI_FSYNC_INT | FSYNC中断 |
| 5 | PLL_RDY_INT | PLL就绪中断 |
| 4 | RESET_DONE_INT | 复位完成中断 |
| 3 | DATA_RDY_INT | 数据就绪中断 |
| 2 | FIFO_THS_INT | FIFO阈值中断 |
| 1 | FIFO_FULL_INT | FIFO满中断 |
| 0 | AGC_RDY_INT | AGC就绪中断 |

**读取清除**: 读此寄存器自动清除中断标志(读清除模式)

#### 4.2.7 APEX数据输出 (只读)

| 寄存器 | 地址 | 内容 |
|--------|------|------|
| APEX_DATA0 (0x31) | STEP_CNT[7:0] | 步数低字节 |
| APEX_DATA1 (0x32) | STEP_CNT[15:8] | 步数高字节 |
| APEX_DATA2 (0x33) | STEP_CADENCE | 步频 |
| APEX_DATA3 (0x34) | ACTIVITY_CLASS | 活动类型 |
| APEX_DATA4 (0x35) | TAP_NUM/TAP_AXIS/TAP_DIR | 敲击检测 |
| APEX_DATA5 (0x36) | DOUBLE_TAP_TIMING | 双击时间间隔 |

**活动类型 (ACTIVITY_CLASS)**:
```
00: Unknown
01: Walk (步行)
10: Run (跑步)
11: Reserved
```

## 5. 初始化与配置

### 5.1 标准初始化流程

```c
// 标准ICM-42688-P初始化流程
void ICM42688_Init(void) {
    // 1. 等待上电稳定
    delay_ms(10);
    
    // 2. 软件复位
    ICM_WriteReg(0x11, 0x01);  // DEVICE_CONFIG: 软件复位
    delay_ms(2);
    
    // 3. 检查WHO_AM_I (可选)
    uint8_t whoami = ICM_ReadReg(0x75);  // WHO_AM_I
    if (whoami != 0x00) {  // 实际值需要确认
        // 错误处理
    }
    
    // 4. 配置中断引脚
    ICM_WriteReg(0x14, 0x03);  // INT_CONFIG: 
                               // INT1推挽输出, 高电平有效, 脉冲模式
    
    // 5. 配置FIFO
    ICM_WriteReg(0x16, 0x40);  // FIFO_CONFIG: Stream-to-FIFO模式
    
    // 6. 配置传感器参数
    // 陀螺仪: ±2000dps, 1kHz ODR
    ICM_WriteReg(0x4F, 0x06);  // GYRO_CONFIG0
    
    // 加速度计: ±16g, 1kHz ODR
    ICM_WriteReg(0x50, 0x06);  // ACCEL_CONFIG0
    
    // 7. 使能传感器 (低噪声模式)
    ICM_WriteReg(0x4E, 0x0F);  // PWR_MGMT0:
                               // 温度使能, 陀螺LN, 加速度LN
    
    // 8. 等待传感器稳定
    delay_ms(50);
}
```

### 5.2 低功耗配置示例

```c
// 仅加速度计低功耗模式
void ICM42688_LowPower_Init(void) {
    delay_ms(10);
    
    // 软件复位
    ICM_WriteReg(0x11, 0x01);
    delay_ms(2);
    
    // 加速度计: ±8g, 50Hz ODR, 低功耗模式
    ICM_WriteReg(0x50, 0x09);  // ACCEL_CONFIG0: ±8g, 50Hz
    
    // 配置为低功耗模式
    ICM_WriteReg(0x4D, 0x00);  // INTF_CONFIG1: LP时钟选择
    
    // 仅使能加速度计(LP模式)
    ICM_WriteReg(0x4E, 0x02);  // PWR_MGMT0: ACCEL_MODE=10 (LP)
    
    delay_ms(50);
}
```

## 6. 数据读取

### 6.1 直接读取传感器数据

```c
typedef struct {
    int16_t accel_x;
    int16_t accel_y;
    int16_t accel_z;
    int16_t gyro_x;
    int16_t gyro_y;
    int16_t gyro_z;
    float temperature;
} ICM_Data_t;

void ICM_ReadSensorData(ICM_Data_t* data) {
    uint8_t buffer[14];
    
    // 批量读取所有传感器数据 (从0x1D开始)
    ICM_ReadBurst(0x1D, buffer, 14);
    
    // 解析温度
    int16_t temp_raw = (buffer[0] << 8) | buffer[1];
    data->temperature = (temp_raw / 132.48f) + 25.0f;
    
    // 解析加速度 (2g/4g/8g/16g对应不同换算系数)
    data->accel_x = (buffer[2] << 8) | buffer[3];
    data->accel_y = (buffer[4] << 8) | buffer[5];
    data->accel_z = (buffer[6] << 8) | buffer[7];
    
    // 解析陀螺仪
    data->gyro_x = (buffer[8] << 8) | buffer[9];
    data->gyro_y = (buffer[10] << 8) | buffer[11];
    data->gyro_z = (buffer[12] << 8) | buffer[13];
}

// 换算为物理单位
void ICM_ConvertToPhysical(ICM_Data_t* raw, float* accel_g, float* gyro_dps) {
    // 根据ACCEL_FS_SEL配置换算 (假设±16g)
    float accel_sensitivity = 2048.0f;  // LSB/g for ±16g
    accel_g[0] = raw->accel_x / accel_sensitivity;
    accel_g[1] = raw->accel_y / accel_sensitivity;
    accel_g[2] = raw->accel_z / accel_sensitivity;
    
    // 根据GYRO_FS_SEL配置换算 (假设±2000dps)
    float gyro_sensitivity = 16.4f;  // LSB/dps for ±2000dps
    gyro_dps[0] = raw->gyro_x / gyro_sensitivity;
    gyro_dps[1] = raw->gyro_y / gyro_sensitivity;
    gyro_dps[2] = raw->gyro_z / gyro_sensitivity;
}
```

## 7. APEX运动处理引擎

### 7.1 APEX功能概述

APEX (Advanced pedometer and Event detection) 是ICM-42688-P内置的DMP (Digital Motion Processor)，支持：

- **计步器 (Pedometer)**: 自动步数统计
- **倾斜检测 (Tilt Detection)**: 设备方向变化检测
- **敲击检测 (Tap Detection)**: 单/双击检测
- **唤醒/睡眠检测 (Raise to Wake/Sleep)**: 抬手亮屏
- **运动唤醒 (Wake on Motion)**: 运动触发唤醒
- **显著运动检测 (Significant Motion Detection)**: 大幅度运动检测

### 7.2 计步器配置

```c
void ICM42688_EnablePedometer(void) {
    // 1. 切换到Bank 4
    ICM_WriteReg(0x76, 0x04);  // REG_BANK_SEL = Bank 4
    
    // 2. 配置计步器参数
    ICM_WriteReg(0x40, 0x0A);  // APEX_CONFIG1: LOW_ENERGY_AMP_TH_SEL=10
    ICM_WriteReg(0x41, 0x58);  // APEX_CONFIG2: 
                               // PED_AMP_TH_SEL=8, PED_STEP_CNT_TH_SEL=5
    ICM_WriteReg(0x42, 0x12);  // APEX_CONFIG3:
                               // PED_STEP_DET_TH_SEL=2, PED_HI_EN_TH_SEL=1
                               // PED_SB_TIMER_TH_SEL=4
    ICM_WriteReg(0x48, 0x00);  // APEX_CONFIG9: SENSITIVITY_MODE=0
    
    // 3. 切换回Bank 0
    ICM_WriteReg(0x76, 0x00);
    
    // 4. 使能DMP
    ICM_WriteReg(0x4B, 0x40);  // SIGNAL_PATH_RESET: DMP_INIT_EN=1
    delay_ms(1);
    
    // 5. 配置加速度计 (50Hz, LP模式)
    ICM_WriteReg(0x50, 0x09);  // ACCEL_CONFIG0: 50Hz
    ICM_WriteReg(0x4E, 0x02);  // PWR_MGMT0: ACCEL LP模式
    
    // 6. 配置DMP ODR
    ICM_WriteReg(0x56, 0x02);  // APEX_CONFIG0: DMP_ODR = 50Hz
    
    // 7. 使能计步器中断
    ICM_WriteReg(0x76, 0x04);  // Bank 4
    ICM_WriteReg(0x4D, 0x20);  // INT_SOURCE6: STEP_DET_INT1_EN
    ICM_WriteReg(0x76, 0x00);  // Bank 0
    
    // 8. 等待DMP初始化
    delay_ms(50);
    
    // 9. 启动计步器
    ICM_WriteReg(0x56, 0x04);  // APEX_CONFIG0: PED_ENABLE=1
}

uint16_t ICM42688_ReadStepCount(void) {
    uint8_t low = ICM_ReadReg(0x31);   // APEX_DATA0
    uint8_t high = ICM_ReadReg(0x32);  // APEX_DATA1
    return (high << 8) | low;
}
```

### 7.3 敲击检测配置

```c
void ICM42688_EnableTapDetection(void) {
    // 1. 配置加速度计 (500Hz)
    ICM_WriteReg(0x50, 0x0F);  // ACCEL_CONFIG0: 500Hz
    ICM_WriteReg(0x52, 0x44);  // GYRO_ACCEL_CONFIG0
    ICM_WriteReg(0x53, 0x22);  // ACCEL_CONFIG1
    
    // 2. 配置敲击参数 (Bank 4)
    ICM_WriteReg(0x76, 0x04);
    ICM_WriteReg(0x47, 0x23);  // TAP_TMAX=2, TAP_TAVG=3, TAP_TMIN=3
    ICM_WriteReg(0x46, 0xA2);  // TAP_MIN_JERK_THR=17, TAP_MAX_PEAK_TOL=2
    ICM_WriteReg(0x76, 0x00);
    
    // 3. 使能中断
    ICM_WriteReg(0x76, 0x04);
    ICM_WriteReg(0x4D, 0x01);  // INT_SOURCE6: TAP_DET_INT1_EN
    ICM_WriteReg(0x76, 0x00);
    
    // 4. 等待并启动
    delay_ms(50);
    ICM_WriteReg(0x56, 0x02);  // APEX_CONFIG0: TAP_ENABLE=1
}

void ICM42688_ReadTapInfo(uint8_t* num, uint8_t* axis, uint8_t* dir) {
    uint8_t data = ICM_ReadReg(0x35);  // APEX_DATA4
    *num = (data >> 3) & 0x03;   // TAP_NUM
    *axis = (data >> 1) & 0x03;  // TAP_AXIS
    *dir = data & 0x01;          // TAP_DIR
}
```

## 8. FIFO配置与使用

### 8.1 FIFO模式

| 模式 | 值 | 说明 |
|------|-----|------|
| Bypass | 0x00 | FIFO禁用，直接读取数据寄存器 |
| Stream-to-FIFO | 0x01 | 循环缓冲，覆盖旧数据 |
| STOP-on-FULL | 0x02/0x03 | 满后停止写入，防止覆盖 |

### 8.2 FIFO配置寄存器

```c
// FIFO_CONFIG1 (0x5F)
// Bit 7: FIFO_RESUME_PARTIAL_RD
// Bit 6: FIFO_WM_GT_TH
// Bit 5: FIFO_HIRES_EN (20-bit模式)
// Bit 4: FIFO_TMST_FSYNC_EN
// Bit 3: FIFO_TEMP_EN
// Bit 2: FIFO_GYRO_EN
// Bit 1: FIFO_ACCEL_EN

// FIFO_CONFIG2 (0x60) 和 FIFO_CONFIG3 (0x61)
// 设置FIFO水印阈值 (12-bit值)
```

### 8.3 FIFO数据包格式

**标准16-bit数据包 (8 bytes)**:
```
[Header][Accel X][Accel Y][Accel Z][Gyro X][Gyro Y][Gyro Z]
  1byte    2bytes   2bytes   2bytes   2bytes   2bytes   2bytes
```

**20-bit高精度数据包 (16 bytes)**:
```
[Header][Accel X][Accel Y][Accel Z][Gyro X][Gyro Y][Gyro Z][Ext]
  1byte   3bytes   3bytes   3bytes   3bytes   3bytes   3bytes   1byte
```

### 8.4 FIFO读取示例

```c
void ICM42688_FIFO_Init(void) {
    // 配置FIFO使能加速度计和陀螺仪数据
    ICM_WriteReg(0x5F, 0x0E);  // FIFO_CONFIG1: 
                                // TEMP_EN=1, GYRO_EN=1, ACCEL_EN=1
    
    // 设置水印阈值 (例如100个样本)
    ICM_WriteReg(0x60, 100);   // FIFO_WM[7:0]
    ICM_WriteReg(0x61, 0);     // FIFO_WM[11:8]
    
    // 配置FIFO模式
    ICM_WriteReg(0x16, 0x01);  // FIFO_CONFIG: Stream-to-FIFO
    
    // 配置中断
    ICM_WriteReg(0x63, 0x04);  // INT_SOURCE0: FIFO_THS_INT1_EN
}

uint16_t ICM42688_FIFO_GetCount(void) {
    uint8_t high = ICM_ReadReg(0x2E);  // FIFO_COUNTH
    uint8_t low = ICM_ReadReg(0x2F);   // FIFO_COUNTL
    return (high << 8) | low;
}

void ICM42688_FIFO_Read(int16_t* accel, int16_t* gyro) {
    // 读取FIFO数据 (7字节: 1 header + 6 data)
    uint8_t data[7];
    
    for (int i = 0; i < 6; i++) {
        data[i] = ICM_ReadReg(0x30);  // FIFO_DATA
    }
    
    // 解析Header判断数据类型
    uint8_t header = data[0];
    
    if (header & 0x04) {  // Accel数据可用
        accel[0] = (data[1] << 8) | data[2];
        accel[1] = (data[3] << 8) | data[4];
        accel[2] = (data[5] << 8) | data[6];
    }
    
    // 继续读取陀螺仪数据...
}
```

## 9. 中断配置

### 9.1 中断引脚配置

```c
void ICM42688_INT_Config(void) {
    // INT_CONFIG (0x14)
    uint8_t config = 0x00;
    
    // INT1配置
    config |= 0x01;  // INT1推挽输出
    config |= 0x02;  // INT1高电平有效
    // config |= 0x04;  // INT1锁存模式 (否则脉冲模式)
    
    // INT2配置
    config |= 0x08;  // INT2高电平有效
    config |= 0x20;  // INT2推挽输出
    
    ICM_WriteReg(0x14, config);
}
```

### 9.2 中断源映射

**INT1中断源 (INT_SOURCE0 0x65, INT_SOURCE1 0x66)**:
- Bit 5: UI_FSYNC_INT1_EN
- Bit 4: PLL_RDY_INT1_EN
- Bit 3: RESET_DONE_INT1_EN
- Bit 2: UI_DRDY_INT1_EN (数据就绪)
- Bit 1: FIFO_THS_INT1_EN (FIFO阈值)
- Bit 0: FIFO_FULL_INT1_EN (FIFO满)

**INT_SOURCE1 (0x66) 扩展中断**:
- Bit 3: SMD_INT1_EN (显著运动检测)
- Bit 2: WOM_Z_INT1_EN
- Bit 1: WOM_Y_INT1_EN
- Bit 0: WOM_X_INT1_EN

### 9.3 数据就绪中断示例

```c
void ICM42688_EnableDataReadyINT(void) {
    // 配置INT1引脚
    ICM_WriteReg(0x14, 0x03);  // 推挽, 高电平有效
    
    // 使能数据就绪中断
    ICM_WriteReg(0x65, 0x04);  // INT_SOURCE0: UI_DRDY_INT1_EN
}

// 在中断服务程序中
void ICM_INT1_IRQHandler(void) {
    uint8_t status = ICM_ReadReg(0x2D);  // 读取INT_STATUS清除中断
    
    if (status & 0x08) {  // DATA_RDY_INT
        ICM_ReadSensorData(&sensor_data);
    }
}
```

## 10. 代码示例

### 10.1 完整初始化与读取 (HAL库示例)

```c
#include "stm32f1xx_hal.h"

#define ICM_I2C_ADDR 0xD0  // 7-bit: 0x68, 写地址

I2C_HandleTypeDef hi2c1;

// I2C底层读写函数
static void ICM_WriteReg(uint8_t reg, uint8_t data) {
    uint8_t buf[2] = {reg, data};
    HAL_I2C_Master_Transmit(&hi2c1, ICM_I2C_ADDR, buf, 2, 100);
}

static uint8_t ICM_ReadReg(uint8_t reg) {
    uint8_t data;
    HAL_I2C_Master_Transmit(&hi2c1, ICM_I2C_ADDR, &reg, 1, 100);
    HAL_I2C_Master_Receive(&hi2c1, ICM_I2C_ADDR, &data, 1, 100);
    return data;
}

static void ICM_ReadBurst(uint8_t reg, uint8_t* data, uint16_t len) {
    HAL_I2C_Master_Transmit(&hi2c1, ICM_I2C_ADDR, &reg, 1, 100);
    HAL_I2C_Master_Receive(&hi2c1, ICM_I2C_ADDR, data, len, 100);
}

// 完整初始化
uint8_t ICM42688_Init(void) {
    HAL_Delay(10);
    
    // 软件复位
    ICM_WriteReg(0x11, 0x01);
    HAL_Delay(2);
    
    // 配置电源管理
    ICM_WriteReg(0x4E, 0x00);  // 初始全部关闭
    HAL_Delay(1);
    
    // 配置陀螺仪: ±2000dps, 1kHz
    ICM_WriteReg(0x4F, 0x06);
    
    // 配置加速度计: ±16g, 1kHz
    ICM_WriteReg(0x50, 0x06);
    
    // 配置滤波器 (可选)
    ICM_WriteReg(0x51, 0x56);  // GYRO_CONFIG1
    ICM_WriteReg(0x52, 0x56);  // GYRO_ACCEL_CONFIG0
    
    // 使能传感器 (LN模式)
    ICM_WriteReg(0x4E, 0x0F);  // 温度+陀螺+加速度
    
    HAL_Delay(50);
    
    return 0;
}

// 读取传感器数据
void ICM42688_ReadData(float* accel_g, float* gyro_dps, float* temp) {
    uint8_t buf[14];
    ICM_ReadBurst(0x1D, buf, 14);
    
    // 温度
    int16_t temp_raw = (buf[0] << 8) | buf[1];
    *temp = (temp_raw / 132.48f) + 25.0f;
    
    // 加速度 (假设±16g)
    int16_t accel[3];
    accel[0] = (buf[2] << 8) | buf[3];
    accel[1] = (buf[4] << 8) | buf[5];
    accel[2] = (buf[6] << 8) | buf[7];
    
    // 陀螺仪 (假设±2000dps)
    int16_t gyro[3];
    gyro[0] = (buf[8] << 8) | buf[9];
    gyro[1] = (buf[10] << 8) | buf[11];
    gyro[2] = (buf[12] << 8) | buf[13];
    
    // 转换为物理单位
    const float accel_scale = 1.0f / 2048.0f;  // ±16g
    const float gyro_scale = 1.0f / 16.4f;      // ±2000dps
    
    for (int i = 0; i < 3; i++) {
        accel_g[i] = accel[i] * accel_scale;
        gyro_dps[i] = gyro[i] * gyro_scale;
    }
}
```

### B. 故障排除

| 问题 | 可能原因 | 解决方法 |
|------|----------|----------|
| I2C通信失败 | 地址错误 | 检查AP_AD0引脚状态 |
| | 上拉电阻缺失 | I2C总线需10kΩ上拉 |
| 数据全为0或-1 | 传感器未使能 | 检查PWR_MGMT0寄存器 |
| | 模式切换太快 | 模式切换后等待200µs |
| 温度读数异常 | 未使能温度传感器 | PWR_MGMT0 bit5=0 |
| APEX功能不工作 | DMP未初始化 | 确保DMP_INIT_EN=1并等待50ms |
| | 加速度计未使能 | APEX需要加速度计数据 |
